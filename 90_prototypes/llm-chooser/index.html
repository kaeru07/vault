全 LLM 早見表

Claude（Anthropic）

長文コード Agent CLI Vault連携

Claude Code が CLI で動く。長文コード生成・複雑なリファクタ・複数ファイル横断の作業に強い。Vault / progress / Obsidian と相性良。

Codex（OpenAI）

短い指示 Web 初回応答速

Codex は短い指示への初回応答が速い。Web UI 中心。Agent SDK 経由でも使えるが Claude Code ほど成熟していない（個人観察）。

Gemini（Google）

長文入出力 マルチモーダル AI Studio無料層

Gemini は超長文入出力（数十万トークン）が強み。AI Studio に無料層あり。マルチモーダル（画像 / 音声 / 動画）も扱える。

ローカル LLM（Ollama / llama.cpp）

プライバシー API課金ゼロ オフライン

ローカル LLM はネット不要 + API 課金ゼロ。プライバシー要件が高い用途・GPU/CPU 性能テスト用に最適。精度は商用 LLM より劣る。
