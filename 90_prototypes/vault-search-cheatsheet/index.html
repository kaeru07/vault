iPhone Obsidian の検索 Tips

  • 日本語ファイル名はヒットしないことがある → 英数字キーワード（上表）優先
  • クイックスイッチャー でファイル名検索（最速）
  • 検索 アイコンで全文検索（重い場合あり）
  • tag:#tagname でタグ検索（frontmatter tags 対応）
  • "フレーズ" でフレーズ完全一致
  • path:05_monetization でフォルダ絞り込み（Obsidian 1.0+）

GitHub Web の検索 Tips

  • リポジトリ画面の / でファイル名検索（一番速い）
  • 右上の検索バーで全文検索（インデックス未完了だと遅れる）
  • filename:candidate でファイル名フィルタ
  • path:20_reviews でフォルダフィルタ
  • extension:md で md のみ
  • blame で「いつ誰がこの行を書いたか」

iPhone と GitHub で結果が違うとき

  • iPhone でだけ見えない → obsidian-sync-vault 側に未反映の可能性 → ChatGPT に「逆反映ルール再実行」と依頼
  • GitHub でだけ見えない → obsidian-vault 側に未 push → Claude Code に「GitHub 反映」と依頼
  • HTML / JSON が iPhone で見えない → ob sync は Markdown のみ同期。HTML / JSON は GitHub Web で見る
  • candidate を見つけられない → scenarios で検索（README.md が candidate 全件中継）
