# Claude Code プロンプト — 標準版

機能追加・画面改修・データ追加など、ある程度範囲のある作業向け。

## テンプレ

```
対象:
<アプリパス>

目的:
- <達成したいこと>

現状の課題:
- <なぜ変えるか>

改善内容:
1. <項目1>
2. <項目2>
3. <項目3>

禁止:
- <壊してはいけない既存仕様>
- <スコープ外>

実行:
1. npm run typecheck
2. npm run build（または render など）
3. <該当する動作確認>

完了条件:
- <Done の条件>

報告形式:
【実施内容】
【変更ファイル】
【検証結果】
【未対応】
【次にやるべきこと】
```

## 使うシーン

- 新機能の追加
- データ構造の小変更（optional フィールド追加など）
- 複数ファイルにまたがる改修

## 注意

- 「何を直すか / どこを触るか / どこを触らないか / 完了条件 / 検証方法」を着手前に明示
- 既存仕様の破壊的変更は明確に禁止に書く
- 検証セット（typecheck + build + 動作確認）を完了条件に含める
