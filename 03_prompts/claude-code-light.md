# Claude Code プロンプト — 軽量版

UI 微修正・文言修正・1〜3 ファイルの単発作業向け。短く渡す。

## テンプレ

```
対象: <アプリ名 / 該当ファイルパス>
目的: <1〜2 行で目的>
変更:
- <変更1>
- <変更2>
禁止:
- <触ってほしくない範囲>
検証:
- npm run typecheck
- 該当画面の動作確認
完了報告:
- 変更ファイル / 検証結果 / 残課題を箇条書きで
```

## 使うシーン

- 文言修正・コピー差し替え
- 軽い UI バグ修正
- 単発の typo / lint 修正

## 注意

- 軽量版でも `architect` 5 項目（何を直すか / どこを触るか / どこを触らないか / 完了条件 / 検証方法）は最低限を着手前に整理
- 検証なしで「完了」と報告させない
