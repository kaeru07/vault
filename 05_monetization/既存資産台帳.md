- どこを調べたか: `company/apps/` 実ディレクトリ、`apps/app-index.md`、各アプリ README / package.json / vercel.json の有無 / 2026-05-18 / n=15リポジトリ
- 情報源種別: 一次（ローカルリポジトリ実体）+ 二次（app-index.md の記載）
- **未検証点**: 各アプリの「実際のストア/Vercel 本番公開状態」はライブ確認していない。app-index.md と vercel.json 有無からの**推定**。公開状況列に「未確認/推定」と明記
- 再現性: 高（同ディレクトリ・同 index で再現）
