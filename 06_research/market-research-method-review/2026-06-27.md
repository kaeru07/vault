# 市場調査方法レビュー 2026-06-27

## 今日使った調査方針
収益化しやすさ、短期リリース可能性、Claude Code / Codex で量産しやすいかを優先して見た。  
対象は AI ニュース、AI ツール、MCP / AI Agent、ショート動画、学習アプリ、広告収益系に重きを置いた。  
巨大 AI チャット本体ではなく、周辺ワークフロー、権限管理、動画制作、学習支援へ寄せた。  
日本 / 海外両方の需要を見る方針だったが、今日は海外情報源への比重が大きかった。

## 良かった点
- 公式情報、論文、二次情報を区別し、根拠URLの強弱を明示できた。
- AI Agent / MCP / Computer Use / Codex 利用拡大を、個人開発の収益化候補へ接続できた。
- 「AIチャット本体ではなく、狭い作業完了ツールに寄せる」という判断は方針に合っていた。
- 品質チェックで、昨日との差分未確認、公式未取得、推測部分を明示できた。
- ショート動画、Chrome拡張、業種特化テンプレなど、短期実装しやすい案を複数出せた。

## 弱かった点
- 国内 App Store / Google Play の実ランキング、課金画面、価格比較が不足した。
- TikTok / YouTube Shorts の実アカウント、再生数、収益導線の確認ができていない。
- GPT-5.6 や Windows Intelligent Terminal など、公式未確認の二次情報を AI ツール調査に含めた。
- 競合の価格、無料枠、売上推定が薄く、収益化判断が「需要ありそう」に寄りすぎた。
- 昨日分との比較ができず、Codex、MCP、CapCut など定番テーマの重複率を測れなかった。

## 情報源の偏り
今日カバーした情報源は、OpenAI / Anthropic / Google / GitHub 公式、arXiv、App Store、Google Play、Perplexity、CapCut、The Verge、Axios、Business Insider、WSJ、Tom's Hardware など。  
不足していた情報源は、国内 App Store ランキング、国内 Google Play ランキング、TikTok 内検索、YouTube Shorts 実例、Sensor Tower 等の売上推定、X / Reddit / Product Hunt のユーザー反応。  
AI 開発者向け情報は厚かったが、一般ユーザーが実際に課金しているアプリや動画導線の一次観察が弱かった。

## 収益化判断の精度
収益化ヒントは方向性としては近いが、精度は中程度。  
「業種特化テンプレ」「Chrome拡張の小型委任タスク」「CapCut自動化教材」は短期実装と販売導線が比較的明確だった。  
一方で「AIブラウザ用タスク監査メモ」「MCPツール棚卸しダッシュボード」は需要仮説は強いが、個人・中小企業が今すぐ払う金額や導入経路が未確認。  
作れそうなアプリ案は実装可能性を見ているが、価格、競合課金、獲得チャネルまで落とし込めていない。

## 見落とした可能性
- 国内ユーザー向けの AI 学習アプリ、資格学習、英語学習の課金上位アプリ。
- AI 動画テンプレを販売している個人・小規模事業者の実例と販売単価。
- Chrome Web Store / Shopify App Store / Notion Marketplace など、個人開発者が販売しやすい市場。
- AI Agent セキュリティを個人向けではなく、開発チーム向け月額ツールとして売る導線。

## 次回改善すること
- 国内 App Store / Google Play で AI 関連の上位課金アプリを各3件確認し、価格、無料枠、レビュー不満を記録する。
- TikTok / YouTube Shorts で AI生成動画アカウントを3件取り、投稿頻度、再生数、プロフィール導線、販売先を表にする。
- AI ツール調査では、公式URLが取れない新モデル・新製品は「未確認枠」に分離する。
- 競合アプリごとに、月額価格、年額価格、無料枠、アップセル導線を最低1行で記録する。
- 昨日分 Markdown と比較し、重複テーマと新規テーマを明示する。

## market-research-method.md への反映案
軽微な反映案: AI ツール調査では公式未確認の新製品・新モデルを「未確認枠」として分離する。  
大きな変更案: 日次調査に「競合価格表」と「動画アカウント実例表」を必須化する。  
大きな変更案: 調査対象の一部を入れ替え、Chrome Web Store / Shopify App Store / Notion Marketplace を定期観測対象に加える。
