# 市場調査方法レビュー 2026-07-05

## 今日使った調査方針
収益化しやすさ、短期リリース可能性、Claude Code / Codex で量産しやすいかを最優先に見た。
対象は AI ニュース、AI ツール、MCP、AI Agent、App Store / Google Play、AI 自動化に重きを置いた。
特に Google Play の会話型検索、MCP / Agent 観測、AI動画生成、AI開発コスト削減を収益化候補として扱った。
一方で、麻雀・将棋・学習アプリ・広告収益系は方針上の対象だが、今日の深掘りは弱かった。

## 良かった点
- Google Play の Gemini 会話型検索を、単なるニュースではなく AI検索ASO、説明文改善、B2B小ツールに接続できた。
- MCP / Agent 領域を「接続」ではなく、権限管理、監査ログ、実行観測という収益化に近い層で整理できた。
- AIツール調査では konsistent、Vercel Agent Runs、GitHub Copilot Session Streaming など公式URL中心で確認できた。
- 収益化ヒントが、月額SaaS、テンプレ販売、比較サイト、フリーミアムに分かれており、実験単位に落としやすかった。
- 各ファイルで partial 理由を明記し、未確認情報と推測を分けようとしていた。

## 弱かった点
- App Store / Google Play の当日ランキングを直接確認できず、市場調査の根拠が報道と推測に寄った。
- AIニュースで OpenAI / Anthropic / Google の一次公式情報が不足し、重要ニュースの信頼度が弱かった。
- 麻雀、将棋、学習アプリ、広告収益系の具体的な競合・課金・ランキング観測が薄かった。
- 「ユーザー不満」が未確認推測に寄っており、レビュー、SNS、Reddit、ストア口コミなどの実声が足りなかった。
- AI動画生成は収益化余地を示したが、実際の価格、透かし、商用利用条件、主要アプリ比較が未完了だった。

## 情報源の偏り
今日カバーした情報源は、Vercel / GitHub 公式 changelog、Tech 系報道、Business Insider、AP、Times of India、Economic Times / Bloomberg、arXiv が中心だった。
不足していた情報源は、App Store / Google Play の当日ランキング、実アプリのレビュー、開発者ブログ、公式ドキュメント一次情報、日本語圏SNS、TikTok / Shorts の実トレンドである。
特に AI ツール調査は Vercel / GitHub に強く偏り、独立系AIツール、モバイルアプリ、消費者向け生成AIアプリの観測が少なかった。
市場調査は「AIエージェント運用」へ寄り、方針にある麻雀・将棋・学習・広告収益アプリの定点観測が不足した。

## 収益化判断の精度
収益化に近かった案は、AI検索ASOチェッカー、AI動画プロンプト台本メーカー、MCP操作ログビューア、konsistent 導入支援である。
特に AI検索ASO は Google Play の発見導線変化と小規模開発者の課題がつながっており、低コストで検証しやすい。
一方、AIアプリ比較サイトは競合が多く、SEOや更新コストが重いため、収益化までの距離をやや楽観的に見ている。
MCP操作ログ・危険操作ブロックは需要の方向性は良いが、個人向けにどこまで支払い意思があるか未検証である。
全体として、アイデアの方向は収益化に近いが、価格、顧客、流入経路、実レビュー根拠の確認が不足している。

## 見落とした可能性
- App Store / Google Play 上位の実アプリが、AI検索前提でどのようなタイトル・説明・スクリーンショットを使っているか。
- 麻雀・将棋・学習アプリで、AI要約、復習、棋譜・牌譜レビューが既に課金されている実例。
- AI動画生成アプリの無料枠制限、透かし、商用利用条件、テンプレ販売との相性。
- 日本語圏の個人開発者が実際に困っている ASO、AI開発費、Agent監査、レビュー対応の具体的な不満。

## 次回改善すること
- App Store / Google Play の当日ランキング代替ソースを最低1つ確認し、AI系・学習系・ゲーム学習系アプリを実名で記録する。
- AI動画生成アプリを3件に絞り、月額価格、無料枠、透かし、商用利用、Shorts/TikTok向け出力を表で比較する。
- ユーザー不満は最低3件、ストアレビューまたはSNS由来として根拠付きで拾う。
- OpenAI / Anthropic / Google 関連ニュースは、二次報道を書く前に公式URLの有無を確認し、未取得なら重要度を下げる。
- 麻雀・将棋・学習・広告収益系のうち最低1ジャンルを毎回必ず1段落だけ観測対象に入れる。
