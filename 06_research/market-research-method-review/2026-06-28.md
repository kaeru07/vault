# 市場調査方法レビュー 2026-06-28

## 今日使った調査方針
収益化しやすさ、短期リリース可能性、Claude Code / Codex で量産しやすい領域を優先した。  
対象は App Store / Google Play、AIニュース、AIツール、MCP、AI Agent、AI自動化を中心に見た。  
特に ChatGPT / Claude / Perplexity など上位AIアプリの不満と、Codex / Gemini / MCP 系のエージェント化を重視した。  
一方で Shorts / TikTok、国内ストア、個人開発市場の直接観察は弱かった。

## 良かった点
- App Store / Google Play の上位AIアプリから、履歴消失、ログイン不具合、制限過多など個人開発の入口になりそうな不満を抽出できた。
- AIニュースでは Codex 利用分析、Gemini Computer Use、Claude 障害など、個人開発の実装判断に近い話題を拾えた。
- AIツール調査では MCP Apps、Arcade.dev、OpenClaw など、AIエージェントを商品化する際の基盤側まで見られた。
- 各レポートに品質チェック、未取得情報、次回再調査対象があり、調査の弱点を後続作業に残せていた。
- 収益化ヒントが「汎用AI本体」ではなく、履歴保全、レビュー分析、学習補助、業務チェックリストに寄せられていた。

## 弱かった点
- SNS直接観察、Product Hunt、Reddit、国内ストアランキングが未取得で、実需要の熱量確認が不足した。
- 価格、無料枠、従量課金、制限内容の比較が浅く、収益化判断に必要な単価感が弱かった。
- AIツール調査で「新規AIツール」として扱った一部項目に、当日公開かどうか未確認のものが混ざっていた。
- ChatGPT / Claude / Perplexity / Codex / MCP に寄りすぎており、麻雀、将棋、学習アプリ、広告収益系の具体例が薄かった。
- 昨日との差分が未取得のため、継続テーマと本当に新しい発見の切り分けが不十分だった。

## 情報源の偏り
今日カバーした情報源は、App Store、Google Play、OpenAI公式、Gemini API changelog、Claude Status、arXiv、The Verge、TechRadar、WSJ、Tom's Hardware など。  
公式情報と技術系二次情報は比較的多く、AIエージェント、MCP、AI coding 周辺には厚みがあった。  
不足していた情報源は、TikTok / YouTube Shorts、Reddit、Product Hunt、国内 App Store / Google Play、個人開発者の売上事例、価格ページの一次情報。  
結果として、技術トレンドは拾えたが、消費者向けアプリの需要熱量と実際の課金行動はまだ弱い。

## 収益化判断の精度
収益化ヒントは、履歴保全、AIレビュー不満チェッカー、学習補助、業務チェックリストなど、短期試作しやすい方向に寄っており大きくは外れていない。  
ただし、月額 $5-$15、クレジット課金、広告収益などの提案は、競合価格やユーザー獲得単価の裏取りが足りない。  
「作れそうなアプリ案」は実装可能性は高いが、実際にお金を払う顧客像、流入経路、継続利用理由の検証が不足している。  
現時点の精度は中程度で、アイデア発見としては有効だが、着手判断には価格・流入・競合密度の追加調査が必要。

## 見落とした可能性
- 国内ユーザー向けの AI 学習アプリ、AI議事録、AIチャット補助のランキングとレビュー不満。
- 麻雀・将棋など、方針に含まれる趣味特化ジャンルでの AI添削・復習・広告収益アプリ。
- Product Hunt や Reddit に出ている、小規模AIツールのローンチ反応と価格帯。
- AIエージェント系ツールの法人向けではなく、個人・小規模事業者が実際に払える価格帯。

## 次回改善すること
- TikTok / YouTube Shorts / Reddit のうち最低1つで「AI agent」「AI note taker」「AI study app」の不満を直接確認する。
- 日本の App Store / Google Play で Productivity、Education、仕事効率化系のAIアプリを最低10件確認する。
- ChatGPT、Claude、Perplexity、Jules、Codex、主要AI議事録ツールの価格、無料枠、制限を表で比較する。
- 昨日分Markdownと比較し、新規発見、継続観察、重複を明示的に分ける。
- 麻雀、将棋、学習アプリ、広告収益系から最低1カテゴリを選び、AI応用と収益化導線を具体的に見る。
