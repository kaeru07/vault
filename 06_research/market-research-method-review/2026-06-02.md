# 市場調査方法レビュー 2026-06-02

## 今日使った調査方針
収益化しやすく、短期リリース可能で、Codex / Claude Code で量産しやすいテーマを優先した。  
対象は AI ニュース、AI ツール、MCP / AI Agent、App Store / Google Play、ショート動画、学習アプリ周辺に重きを置いた。  
特に「AIモデル本体」ではなく、投稿作業短縮、ランキング監視、MCPテンプレ、業務自動化のような周辺ツール化しやすい領域を見た。  
動画化しやすさと、個人開発で試作から有料note・ミニSaaSへつなげやすいかも判断軸に入れた。

## 良かった点
- OpenAI on AWS、Claude Code dynamic workflows、Google Managed Agents、AWS MCP Server など、公式情報を中心に AI Agent / MCP 系の変化を拾えた。
- 市場調査、AIニュース、AIツールの3本で「AI開発エージェント化」「MCP経由の業務接続」「ショート動画支援」が一貫して見えていた。
- 収益化ヒントが、AIチャット単体ではなく、ランキング監視、台本生成、MCP導入テンプレ、AWS運用自動化など実装可能な補助ツールに寄っていた。
- 各レポートに品質チェックと取得状況があり、未取得・推測・二次情報の区別がある程度できていた。
- 個人開発への影響を、作るものだけでなく利用料、クレジット課金、テスト・CI・運用コストまで含めて見られていた。

## 弱かった点
- App Store 公式ランキング、TikTok / YouTube Shorts の公式収益条件、昨日との差分が未取得で、収益性の根拠が弱い部分が残った。
- AIショート動画の不満やRPM低下は Reddit・二次情報寄りで、実際の投稿者データや公式条件との照合が不足していた。
- 競合アプリの価格、無料枠、課金導線、レビュー不満が表形式で比較されておらず、すぐ作る判断にはまだ粗い。
- AIニュースは大手公式発表に寄り、個人開発者が販売できる小さな実例やマーケットプレイス事例が少なかった。
- Grok Build、Gemini Managed Agents、Antigravity の料金・制限が未確認のまま評価に入っており、導入価値の精度に揺れがある。

## 情報源の偏り
今日カバーした情報源は、OpenAI、Anthropic、Google、AWS、Salesforce、GitHub、Microsoft などの公式発表と、AppBrain、Similarweb、Appshunter、TechCrunch、Android Central、Tom's Guide、AI動画ツール比較記事、Reddit が中心だった。  
不足していた情報源は、App Store 公式または準公式ランキング、Google Play の複数国ランキング、TikTok / YouTube 公式収益条件、各AI動画ツールの公式価格表、実ユーザーのレビュー本文、個人開発者の販売実績、国内市場の具体例。  
特に米国・英語圏のAI開発者向け情報に偏り、日本市場で売れる学習、麻雀、将棋、業務自動化の検証が薄かった。

## 収益化判断の精度
「収益化ヒント」と「作れそうなアプリ案」は、AI動画台本生成、ランキング日次差分、MCP棚卸し、AWS運用自動化など、短期試作しやすいものが多く、収益化への距離は比較的近い。  
一方で、実際に誰がいくら払うかの確認は弱い。特にショート動画支援は需要がありそうでも、既存大手ツールの無料枠やクレジット価格に勝てるかが未検証。  
MCP / Agent 系はB2B受託・導入支援に近いが、個人向けミニSaaSとして成立する価格帯や販売導線はまだ曖昧。  
総合評価は「アイデア発見としては良いが、課金検証前」。次回は価格・無料枠・不満レビュー・購入導線を同時に見る必要がある。

## 見落とした可能性
- AI動画生成ツールの公式価格、クレジット消費、商用利用条件を横並びで見ておらず、実際のコスト差を逃した可能性。
- 国内の学習アプリ、麻雀、将棋、資格学習アプリで AI tutor 化や広告収益化の動きがないかを十分に見ていない。
- Chrome拡張、Notionテンプレ、Gumroad / note / BOOTH など、小さく売れる配布面の調査が不足していた。
- App Store / Google Play のレビュー本文から、ユーザーが実際に困っている反復作業や課金不満を抽出できていない。

## 次回改善すること
- App Store / Google Play のAIチャット、動画編集、教育カテゴリで、上位5件の価格、無料枠、課金導線、レビュー不満を1表にする。
- TikTok Creator Rewards、YouTube Shorts収益化条件、主要AI動画ツールの商用利用条件を公式ページで分けて確認する。
- AI Agent / MCP 系は、公式発表だけでなく「個人が売れるテンプレ、導入代行、監視ツール」に落とせる事例を最低3件探す。
- 昨日分Markdownがある場合は必ず参照し、継続テーマ・新規テーマ・重複テーマを明示する。
- 国内市場として、学習アプリ、麻雀、将棋、業務自動化のうち最低1カテゴリを毎日1つ深掘りする。
