# 市場調査方法レビュー 2026-05-26

## 今日使った調査方針
収益化しやすさ、短期リリース可能性、Claude Code / Codex で量産しやすいことを最優先に見た。  
対象は App Store / Google Play、AIニュース、AIツール、MCP、AI Agent、短尺動画、広告収益系に重きを置いた。  
麻雀・将棋・学習アプリは方針上の対象だが、今日はほぼ深掘りできていない。  
動画化しやすさは TikTok / Shorts / AI広告クリエイティブ領域で一部確認した。

## 良かった点
- Google Play の AI発見導線、TikTok の広告AI機能、MCP / Agent 系を収益化観点で接続できた。
- 「汎用AIチャットへの正面対決は避ける」という競合回避の判断は妥当だった。
- 収益化ヒントが ASO、レビュー分析、広告クリエイティブ、AIコード監査など小さく作れる形に寄っていた。
- AIニュースとAIツール調査が、MCP・セキュリティ・Agent 実行基盤という同じ構造変化を拾えていた。
- 未確認情報には一部「未確認」と明記できており、断定を避ける姿勢は機能した。

## 弱かった点
- App Store / Google Play の実ランキング、売上、レビュー件数などの一次データ確認が薄く、収益判断が推測に寄った。
- 個人開発者の実課金事例、MVP販売実績、Stripe / Gumroad / note / BOOTH などの販売面を十分に見ていない。
- AIニュース側に公式発表ベースの大きな話題が多く、個人が今週作るべき小型プロダクトへの落とし込みがやや粗い。
- 麻雀、将棋、学習アプリなど方針にある既存重点ジャンルとの接続が弱かった。
- 参考URLの一部に誤字や未検証ソースが混じっており、再現性のある調査ログとしては精度不足がある。

## 情報源の偏り
今日カバーした情報源は、Google / OpenAI / Anthropic / AWS などの公式発表、TikTok 公式、Product Hunt 系、Similarweb / AppBrain / AppRank 系、arXiv、技術メディア、Reddit / Hacker News 系の未確認投稿だった。  
不足していた情報源は、App Store / Google Play の実レビュー本文、課金導線スクリーンショット、Sensor Tower / data.ai 系の市場推定、個人開発者の収益公開、広告ライブラリ、YouTube / TikTok の実動画クリエイティブ、国内市場の note / X / Qiita / Zenn 投稿だった。  
特に海外AI公式ニュースに偏り、日本語圏の小規模課金ニーズとモバイルアプリ実需の確認が弱かった。

## 収益化判断の精度
「収益化ヒント」は方向性としては近いが、実際の支払い意欲を確認できたものは少ない。ASO診断、レビュー不満抽出、AI広告台本生成は個人開発向きだが、競合価格、顧客獲得経路、継続課金理由の検証が不足している。  
「作れそうなアプリ案」は実装可能性の評価はできている一方、誰が月額で払うか、初月にどこで売るか、無料ツールとの差分がまだ弱い。  
総合評価は中程度。ネタ出しとしては有効だが、収益化判断としては一次レビュー・価格・販売チャネルの確認が必要。

## 見落とした可能性
- App Store / Google Play の低評価レビューから、AI以外の既存アプリにある明確な課金不満を拾う作業。
- 麻雀、将棋、学習アプリに AI / 短尺動画 / 広告収益を組み合わせる具体案。
- 国内個人開発者が実際に売っている小型ツール、テンプレ、診断サービスの価格帯。
- TikTok / YouTube Shorts の実広告クリエイティブから、どの訴求がAIアプリ課金に近いかを見る作業。

## 次回改善すること
- App Store / Google Play で生成AI・学習・ゲーム補助カテゴリを各5件ずつ見て、レビュー不満と課金導線を表にする。
- TikTok Creative Center または広告ライブラリから、AIアプリ広告の訴求を最低10本確認し、冒頭3秒・CTA・価格訴求を記録する。
- Product Hunt だけでなく、X / note / Zenn / Reddit から個人開発者の実収益・販売報告を最低3件拾う。
- 麻雀・将棋・学習アプリのいずれか1ジャンルを必ず収益化ヒントに接続する。
- 公式AIニュースは「個人が7日以内に作れる派生ツール」に変換できるものだけを上位扱いする。

## market-research-method.md への反映案
軽微な反映案: 次回から「App Store / Google Play の実レビューと課金導線を最低各5件確認する」を調査時チェックに加える。  
大きな変更案: AI公式ニュースの比重を下げ、個人開発者の実収益・販売チャネル確認を独立セクション化する。  
大きな変更案: 麻雀・将棋・学習アプリを毎日ではなく曜日ローテーションで深掘り対象にする。
