# 市場調査方法レビュー 2026-05-26

## 今日使った調査方針
収益化しやすさ、短期リリース可能性、Claude Code / Codex で量産しやすい領域を優先した。  
対象は App Store / Google Play、AI ニュース、AI ツール、MCP、AI Agent、Shorts / TikTok、広告収益系に重きを置いた。  
麻雀・将棋・学習アプリは方針上の対象だが、今日はほぼ深掘りできていない。  
全体として「AIチャット本体」ではなく、ノート化・PDF・広告運用・MCP連携など周辺課金に寄せた。

## 良かった点
- App Store / Google Play の生産性カテゴリから、AIノート・AI検索・PDF系の課金余地を拾えていた。
- Google、Anthropic、OpenAI、GitHub など公式情報を多めに使い、MCP / Agent 周辺の変化を追えていた。
- 「収益化ヒント」と「作れそうなアプリ案」が、短期実装・動画化・テンプレ販売に接続されていた。
- AIニュースとAIツール調査で、Codex / Claude / Gemini / MCP の個人開発への影響を横断できていた。
- 未確認情報には一部「未確認」と明記できており、断定しすぎを避けていた。

## 弱かった点
- ストア上位の具体的な順位変動、レビュー件数、低評価レビュー本文の引用が不足していた。
- 麻雀、将棋、学習アプリなど方針にある個別ジャンルが、実質的に観察対象から外れていた。
- TikTok / Shorts は市場機会として触れたが、実際に伸びている動画、投稿者、訴求、コメント不満の確認が弱い。
- 価格・課金条件が「未確認」のまま残ったツールが多く、導入判断の精度が落ちた。
- 「作れそうなアプリ案」は出ているが、競合数・集客導線・初期販売先までの検証が浅い。

## 情報源の偏り
今日カバーした情報源は、Google / Anthropic / OpenAI / GitHub などの公式発表、TechCrunch / TechRadar などのテックメディア、App Store / Google Play ランキング系サイト、a16z、Better Launch、TikTok World 関連記事が中心だった。  
不足していた情報源は、実際のストアレビュー、Reddit / X / YouTube / TikTok の一次ユーザー反応、Product Hunt の直近ローンチ、個人開発者の売上公開、国内ユーザー向けのアプリランキング、広告運用者コミュニティ。  
特に公式発表と大手メディア寄りで、ユーザー不満と支払い意思の一次情報が弱かった。

## 収益化判断の精度
「AIノート」「PDF差分・要約」「TikTok広告案生成」「MCP導入支援」は、短期実装・課金導線・Codex活用の観点では収益化に比較的近い。  
一方で、実際に誰が月額課金するか、既存競合より何が明確に楽になるか、販売チャネルはどこかの検証が不足していた。  
「MCPサーバー雛形ジェネレーター」は技術的には有望だが、個人が即買う商品というより開発者向け支援・受託・テンプレ販売寄りで、収益化までの距離はやや長い。  
全体評価は中程度。市場の方向性は捉えたが、支払い意思の裏取りが弱い。

## 見落とした可能性
- AIノート系の低評価レビューにある「精度」「要約品質」「録音失敗」「サブスク不満」の具体パターン。
- 麻雀・将棋・学習アプリにAIを足した場合の、広告収益・買い切り・教材販売の可能性。
- 国内向けのLINE連携、Notionテンプレ、スプレッドシート自動化など、日本語圏で売りやすい小型商材。
- TikTok / Shorts の実投稿から見える、AIツール紹介動画のコメント不満と購入導線。

## 次回改善すること
- App Store / Google Play で「AI Note」「PDF AI」「AI Agent」を検索し、上位5件ずつの低評価レビューを最低3件ずつ確認する。
- Product Hunt / Better Launch で直近7日以内のAIツールを見て、ノート・PDF・広告・MCP系の新規参入を分類する。
- TikTok / YouTube Shorts でAIアプリ紹介動画を最低5本確認し、訴求文句、価格、コメント欄の不満を記録する。
- 麻雀・将棋・学習アプリのどれか1領域を必ず1つ選び、AI追加で収益化できる案を最低1件出す。
- 価格が未確認のツールは、公式料金ページまたはヘルプを優先して確認し、「不明」のまま残す件数を減らす。
