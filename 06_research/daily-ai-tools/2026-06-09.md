## 個人開発で活用できそうなもの
- Codex plugins: リサーチ、実装、記事化、リリース作業を役割別pluginに分けると、個人開発の反復作業を標準化しやすい。課金前提。
- Cursor Design Mode: LP、管理画面、学習アプリのUI微修正を自然言語と視覚指定で回せる可能性が高い。価格詳細は未確認。
- GitHub Copilot CLI rubber duck agent: 実装前レビュー、テスト観点洗い出し、PR前セルフレビューに使いやすい。Copilot課金とAI Credits消費に注意。
- Microsoft Foundry Skills: Azure前提だが、MCP resourceとしてスキル化できるため、将来的に収益化用agentテンプレを再利用しやすい。
- Vercel MCP: Vercelのログ、docs、project metadataをAIツールから参照できるため、個人開発アプリのデプロイ調査・障害対応を短縮できる。セルフホストではなくVercel接続型 / https://vercel.com/docs/agent-resources/vercel-mcp
