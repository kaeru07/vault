## 個人開発で活用できそうなもの
- Codex Sites: LP、管理画面、調査レポートWeb化を短時間で作れる可能性が高い。個人開発の検証用デモURL作成に向く。
- GitHub Copilot App: 複数IssueをAIに並列処理させ、個人開発の開発速度を上げる用途に合う。MCP連携が収益化用ワークフローにも使えそう。
- Copilot CLI refresh: cron的な定期プロンプト、レビュー補助、CLI作業の自動化に向く。動画化もしやすい。
- Walrus Memory: AI Agent SaaSを作る場合、ユーザーごとの長期記憶・検証可能履歴の基盤候補。ただし料金と実装成熟度は未確認。
- QVAC SDK TurboQuant: ローカルAIアプリ、学習アプリ、将棋・麻雀解析ツールの端末内推論コスト削減に使える可能性。
