## 記事本文
今日のAIツール棚卸しでは、単体のチャット型ツールよりも「既存の開発・運用面に入り込むエージェント」が目立った。Lovable MCPは、これまでLovable内のチャットで閉じていたアプリ生成を、Claude、Cursor、VS Codeなど外部AIクライアントから呼び出せる方向に広げている。これは個人開発者にとって、アイデア出し、画面生成、DB作成、デプロイを一つの作業列として自動化しやすくなる変化だ。

一方で、GitHub周辺はセキュリティと保守作業の自動化に寄っている。Copilotの `/security-review` はローカルまたは作業中の変更に対して軽量な脆弱性確認を行い、Agentic AutofixはCode scanning alertを起点に、リポジトリ探索、修正、CodeQL再実行、PR作成まで進める。個人開発では大規模なSecurity製品は過剰になりがちだが、広告収益アプリ、SaaS、課金導線を持つ小規模サービスでは、決済・認証・入力処理の初期ミスを潰す価値がある。

Base44 Superagentsは、アプリ生成より一段外側の「運用代行」に近い。Gmail、Slack、CRM、WhatsAppなどをつなぎ、リード対応、週次レポート、予定調整を自然言語で作れる。個人開発の収益化では、作る速度だけでなく、問い合わせ対応、SNS投稿、顧客フォロー、簡易分析をどれだけ自動化できるかが利益率に直結する。ノーコード寄りである分、細かい制御やセルフホスト性は弱いが、売上検証フェーズでは導入価値がある。

Gemini CLIとAI SDKのMCP対応は、より開発者向けの基盤として重要だ。Gemini CLIは無料枠付きのターミナルエージェントとして、CodexやClaude Codeと同じ作業空間に置きやすい。AI SDK v6系はMCPクライアント、ツール承認、型付きツール出力など、ユーザー向けAIエージェントを自前サービスに組み込む部品が揃っている。次の一歩は、すぐ課金可能な小型SaaSを一つ選び、Lovable MCPまたはBase44でプロトタイプを作り、Claude Code/Codexでコード監査と決済導線の堅牢化を行うことだ。

## 個人開発で活用できそうなもの
- Lovable MCP: Claude CodeやCursorから「LP、認証、待機リスト、Supabase風DB、デプロイ」まで一気通貫で試作し、検証用SaaSを短時間で作る用途に向く。
- Base44 Superagents: 個人サービスの問い合わせ一次対応、リード追跡、週次売上レポート、Slack通知をノーコードで作る用途に向く。課金: Starter $16/月 年払い。
- GitHub Copilot Security Review: 決済、認証、フォーム、管理画面まわりの変更前チェックに使える。public preview中は無料枠でも利用可能と公式記載。
- Gemini CLI: 無料枠でターミナル常駐の調査・コード補助に使える。Codexのサブ確認役として、README生成や軽い実装相談に回しやすい。
- Replit Agent: アプリ公開まで含むプロトタイプに向く。課金: Core $20/月 年払い、Pro $95/月 年払い。Starterは無料枠あり。

## MCP/Agent系情報
- Model Context Protocol 2025-06-18仕様: Resources、Prompts、Toolsに加え、Roots、Sampling、Elicitationなどが整理され、ユーザー同意とツール安全性が強調されている。公式: https://modelcontextprotocol.io/specification/2025-06-18
- Lovable MCP: OAuthで接続し、APIキーなしでLovable上のプロジェクト、ファイル、DB、デプロイをAIクライアントから操作可能。公式: https://lovable.dev/mcp
- AI SDK v6 MCP対応: `@ai-sdk/mcp` でHTTP、SSE、stdioのMCP接続を扱える。productionではHTTP transport推奨。公式: https://ai-sdk.dev/docs/ai-sdk-core/mcp-tools
- Base44 Superagents: APIまたはMCP接続に対応すると公式FAQに記載。実装詳細・SDKは本調査では未確認。公式: https://base44.com/superagents
