## 個人開発で活用できそうなもの
- Streamdown: AIチャット、学習アプリ、レビュー支援ツールの回答UIを短時間で高品質化できる。Mermaid・数式・CJK対応があり、日本語学習/技術解説アプリ向き。
- Workflow SDK: メール送信、課金後オンボーディング、長時間AI調査などをdurable workflow化できる。個人SaaSの裏側の失敗耐性に効く。
- Vercel AI Gateway: OpenAI/Anthropic等のモデル切替、fallback、使用量監視をまとめられる。小規模AI SaaSのコスト管理に有用。
- GitHub MCP Registry: 既存MCPサーバー探索に使える。個人開発の自動化エージェントへGit、filesystem、fetch、memory等を足す入口になる。
- OpenAI Codex: issue処理、テスト追加、リファクタ、PRレビューを日次で回せる。課金が必要な場合は費用対効果を要確認。

## Claude Code/Codex連携可能性
- GitHub Copilot Agents: GitHub上でCopilot、Claude、OpenAI Codexなどをissue/PRに割り当ててバックグラウンド実装する連携が可能。URL: https://github.com/features/copilot/
- Claude Code MCP: Google Drive、Jira、Slack、自作ツールをMCP経由でClaude Codeへ接続可能。URL: https://code.claude.com/docs/en/overview
- OpenAI Codex CLI: `npm install -g @openai/codex` または公式インストーラでCLI導入し、ローカルrepo作業に接続可能。URL: https://github.com/openai/codex
- Vercel AI Gateway: Codex/Claude Codeで生成したAIアプリのモデル呼び出しを1つのGatewayに集約できる。URL: https://vercel.com/docs/ai-gateway
- Vercel AI SDK: CodexでNext.js/ReactのAI UIを実装し、AI SDKでstreaming・tool calling・structured outputを組み込む用途に向く。URL: https://github.com/vercel/ai
