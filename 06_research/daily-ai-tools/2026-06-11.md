## 個人開発で活用できそうなもの
- Figma MCP Server: LP、管理画面、アプリUIをFigmaからCodex/Claude Codeへ渡して実装精度を上げられる。個人開発の「デザインから実装」時間短縮に直結。
- OpenAI Codex: 小さなSaaSや学習アプリのIssue単位実装、テスト追加、ドキュメント整備を非同期で投げられる。課金は中程度なので収益化前は利用量管理が必要。
- Cloudflare Agents MCP: 個人開発サービスの管理API、DB操作、通知処理をMCP化してAgentから扱える。Cloudflare Workers前提なら低コストで試せる。
- GitHub Copilot cloud agent: backlogの軽微Issue、依存更新、テスト修正をGitHub上で処理させられる。GitHub Actions minutesとAI creditsの消費管理が必要。
- Claude Code: ローカルCLI、IDE、MCP、CI連携が揃っており、個人開発の継続運用・PRレビュー・定期タスク自動化に使いやすい。

## Claude Code/Codex連携可能性
- Figma MCP Server: Claude Code/CodexにFigmaリンクを渡し、デザイン変数・コンポーネント・Auto Layout文脈を読み込ませて実装。
- Cloudflare Remote MCP: 個人サービスの管理ツールをRemote MCP化し、Claude Code/Codexから安全な範囲で操作。
- OpenAI Agents SDK MCP: Codex外の独自Python AgentからMCP serverを呼び、調査・生成・通知ワークフローを組む。
- GitHub Copilot cloud agent: Codex/Claude Codeで作ったIssue仕様をGitHubに登録し、Copilot cloud agentに実装を委譲。
- Claude Code Agent SDK: Claude Codeのツール・権限・オーケストレーションを使って、個人用の開発Agentを構築。

## MCP/Agent系情報
- MCP公式: AIアプリケーションと外部システムを接続するOSS標準として、Claude、ChatGPT、VS Code、Cursor等の広範な対応を確認 https://modelcontextprotocol.io/docs/getting-started/intro
- Figma MCP Server: Remote MCPでCanvas書き込み、UIからFigma生成、Figma frameからコード生成、Code Connect連携を提供 https://help.figma.com/hc/en-us/articles/32132100833559-Guide-to-the-Figma-MCP-server
- Cloudflare Agents MCP: Remote MCP server、OAuth、Streamable HTTP、権限分割を前提にしたMCP構築ガイドを確認 https://developers.cloudflare.com/agents/model-context-protocol/
- OpenAI Agents SDK MCP: Hosted MCP、Streamable HTTP、stdio、approval flow、tool filteringなどAgent実装向け機能を確認 https://openai.github.io/openai-agents-python/mcp/
