## 個人開発で活用できそうなもの
- Agent Toolkit for AWS: 個人開発アプリをAWSへ出す時、Codex/Claude Codeに最新AWS DocsとIAM制約付き操作を渡せる。無料枠対象サービスと組み合わせれば初期費用を抑えやすい。
- Nuxt MCP Toolkit: 自作SaaSや管理画面に「AIエージェント用のUI付きMCP」を組み込める。Nuxt利用者なら差別化機能として短期実装しやすい。
- GitHub Copilot app: Issue駆動で小さな修正・依存更新・README生成・PR作成を並列化できる。個人開発では月額課金に対して回収できる作業量があるか要検証。
- OpenAI Codex Windows Computer Use: Windowsアプリ、ローカルGUI、ブラウザ確認込みの開発検証をCodexに任せやすくなる。対象地域・対象プランは要確認。
- Salesforce Developer Edition + Hosted MCP: CRM連携アプリやB2B向けAIエージェントの試作に使える。無料枠はあるが商用展開時の価格は未確認。

## Claude Code/Codex連携可能性
- Agent Toolkit for AWS: Codexは `codex plugin marketplace add aws/agent-toolkit-for-aws`、Claude Codeは `/plugin marketplace add aws/agent-toolkit-for-aws` で導入可能。
- Vercel MCP: Codex CLIは `codex mcp add vercel --url https://mcp.vercel.com`、Claude CodeはHTTP MCPとして追加し、デプロイログ解析やDocs検索に使える。
- Salesforce Hosted MCP: Claude Desktop、Cursor、Claude Code等のMCPクライアントからSalesforce orgのデータ・アクションへ接続可能。
- Nuxt MCP Toolkit: 自作NuxtアプリをMCP server化し、Claude/ChatGPT内でHTML UIを返すエージェントツールにできる。
- Gainsight MCP: 企業向けだが、Claude・ChatGPT・Copilot互換ツールからCSデータを参照する連携例として参考になる。
