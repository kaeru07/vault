## 個人開発で活用できそうなもの
- Codex mobile remote access: 外出中にエージェントの質問・差分・テスト結果を確認できるため、個人開発の非同期実装に向く。
- Google Antigravity CLI: Android SDK操作やWeb/Android開発支援が強く、学習アプリ・広告収益アプリの短期試作に使いやすそう。
- GitHub Copilot app: Issue起点で作業を分離しPRまで持っていけるため、小規模SaaSやテンプレアプリ量産の管理面で有用。
- GitHub MCP Server secret scanning: 個人開発でもAPIキー漏洩防止に直結。MCP対応IDE/CLIと組み合わせる価値が高い。
- WebMCP: まだ提案段階だが、将来的にWebアプリ自体をAIエージェント操作可能にする設計指針として有望。

## Claude Code/Codex連携可能性
- GitHub MCP Server: Claude Code/Codex/Copilot CLIからsecret scanningを呼び、コミット前の漏洩検査を自動化。
- WebMCP: CodexやClaude Codeで作ったWebアプリに構造化操作口を用意し、ブラウザエージェント検証を安定化。
- Codex Goal mode: 要件・成功条件を明示して、個人開発アプリの実装から検証まで継続実行。
- Claude Code plugins / financial agent templates: テンプレート構成を参考に、調査・収益化分析・記事生成エージェントを自作。
- GitHub Copilot app / CLI sessions: GitHub issueを入力点にして、Codex/Claude Codeで作った差分をCopilot側でもレビュー・PR化。
