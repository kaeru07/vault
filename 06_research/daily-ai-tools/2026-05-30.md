## 個人開発で活用できそうなもの
- Coder Agents: セルフホスト可能なので、VPSや自宅サーバー上で複数のCodex/Claude系開発Agentを安全に回す基盤候補。個人SaaSの継続改善・Issue処理に向く。
- Twilio MCP Server and Skills: SMS認証、音声AI、通知BotをClaude Code/Codexから組み込みやすい。課金はTwilio従量課金なので、小規模検証から始めやすい。
- CopilotKit: 既存Webアプリに「アプリ内Agent」を埋め込める。チャットUIだけでなく、ユーザー操作を支援する収益化SaaS機能に転用しやすい。
- Google Antigravity 2.0: Android/Google系アプリのプロトタイプを高速に作る用途で有望。ただし移行・料金・日本での利用条件は要確認。
- AWS MCP Server: 個人開発でもAWS運用をAgent化できる可能性があるが、誤操作リスクとクラウド課金リスクがあるため読み取り中心から試すべき。

## Claude Code/Codex連携可能性
- Twilio MCP Server: Claude Code/CodexにMCP接続し、SMS認証・音声通話・顧客通知機能の実装をAgentに任せる。
- AWS MCP Server: AgentにAWSリソースの参照、設定案作成、限定的な操作を委譲する。最初はread-only権限推奨。
- Salesforce Hosted MCP Servers: Claude CodeやCursor等からSalesforce組織データへ接続し、CRM連携アプリの試作に使える。
- Coder Agents API/UI: Codex/Claude系Agentをセルフホスト実行環境で並列起動し、Issue単位で開発を分担する。
- CopilotKit AG-UI: Codexで作ったNext.js/Reactアプリに、Agent操作UIを組み込む候補。
