## 個人開発で活用できそうなもの
- GitHub Copilot SDK: 自作SaaSの「リポジトリ診断」「PR自動レビュー」「オンボーディングbot」に組み込める。Free/BYOKから試せるため初期費用を抑えやすい。
- Cloudflare Agents MCP: 小型のRemote MCP serverを作り、個人開発アプリの外部連携口として販売・公開できる。セルフホストというよりCloudflare上のマネージド運用。
- GitHub Copilot CLI新UI: issue/PRをターミナルで拾って修正依頼できるため、少人数の開発速度改善に直結。課金: Copilot Pro 10 USD/月以上が現実的。
- GitHub Agentic Workflows: 定期的な依存更新、CI失敗調査、ドキュメント同期を自然言語でActions化できる。公開previewなので本番適用は慎重に。
- Vercel AI SDK v6: Next.js製のAIチャット、RAG、生成UI、agent実装の土台として実用的。SDK自体はOSS利用可能、モデルAPI費用は別。

## Claude Code/Codex連携可能性
- Cloudflare Agents MCP: Claude Code/CodexからRemote MCP serverとして呼び出し、DB検索・課金処理・内部API実行を安全に分離。
- GitHub Copilot SDK: 直接Claude Code/Codex用SDKではないが、MCP toolやGitHub App経由でagent runtimeを連携できる可能性あり。
- Vercel AI SDK: CodexでNext.js実装を生成し、AI SDKのtool calling/MCP clientを組み込む流れが作りやすい。
- GitHub Agentic Workflows: Claude Code/Codexでworkflow Markdownを生成し、GitHub Actions上で継続運用する連携が可能。
- Model Context Protocol reference servers: filesystem/git/memory/fetch等をClaude Code/Codexの外部文脈取得に使える。 https://github.com/modelcontextprotocol/servers
