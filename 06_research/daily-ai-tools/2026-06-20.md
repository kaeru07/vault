## 個人開発で活用できそうなもの
- Microsoft Intelligent Terminal: Codex・Claude Code・Geminiをターミナル内で切り替えられるなら、個人開発の実装、ログ調査、修正依頼を1画面に集約できる。
- GitHub Copilot Agent Mode / Spark: IssueからPR草案まで自動化できるため、小規模SaaSや学習アプリの機能追加を短時間で回せる可能性が高い。
- Cursor: 買収・計算資源強化が事実なら、Composer系モデルの品質と速度改善に期待。既存VS Code系開発者には導入しやすい。
- Arcade.dev: 個人開発でもMCPやAIエージェントに外部APIを触らせる場合、認可・監査レイヤーとして参考になる。
- Copilot Cowork: Microsoft 365を使う業務自動化系の副業・B2Bミニツール検証に向くが、個人用途では課金と企業向け制約が重い。
