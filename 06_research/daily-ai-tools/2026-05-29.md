## 個人開発で活用できそうなもの
- Stable Audio 3: ゲーム、ショート動画、学習アプリ向けのBGM・効果音をローカル生成できる可能性が高い。セルフホスト可なので原価管理しやすい。
- Firecrawl on Vercel Marketplace: 競合調査、価格監視、AIニュース収集、RAG用データ収集の基盤に使いやすい。無料枠で検証し、収益化後に $16/月以上へ移行する運用が現実的。
- Claude Opus 4.8: 個人開発では高単価だが、複雑なリファクタ、設計レビュー、長時間のCodex/Claude Code補助に向く。常用ではなく難所だけ使う。
- Base MCP: Web3/DeFi系の個人サービスで、ユーザー承認付きのポートフォリオ確認・スワップ導線をチャットUIに組み込める可能性がある。金融リスクがあるため本番導入は慎重。
- ChatGPT mobile Codex remote access: 外出中にCodexの作業確認・承認・方針変更ができ、個人開発の非同期運用と相性がよい。

## Claude Code/Codex連携可能性
- Claude Opus 4.8 + Claude Code: 大規模コード調査、移行作業、テスト修正を Claude Code の `xhigh` effort や dynamic workflows で実行。
- Vercel AI Gateway: Codexで生成したAI機能に `anthropic/claude-opus-4.8` を差し替え、モデル課金・フェイルオーバーを一元管理。
- Firecrawl: Codex/Claude Code から市場調査・競合ページ取得・RAG更新をMCPまたはAPI経由で自動化。
- Base MCP: Claude/ChatGPT 連携のMCPとして、エージェントがオンチェーン操作案を作り、ユーザーがBase Appで承認する形にできる。
- Alteryx One MCP Server: 個人開発向けでは重いが、企業データ分析案件では Claude/OpenAI エージェントに既存ワークフローを接続する用途がある。
