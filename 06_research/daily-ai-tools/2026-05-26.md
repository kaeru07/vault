## 個人開発で活用できそうなもの
- OpenAI Codex: モバイル承認と Goal mode により、外出中でも実装・レビュー・テスト修正を止めにくい。課金: ChatGPT Plus / Pro 等に含まれるが具体額は地域・プランで要確認。
- Google AI Studio Android app creation: 学習アプリ、麻雀・将棋の小型ユーティリティ、広告収益アプリの MVP 生成に向く。無料枠あり、公開・API 課金は要確認。
- Gemini API Managed Agents: 自動リサーチ、競合調査、SNS 台本生成などのバックエンド Agent 化に向く。課金: 従量課金想定だが 2026-05-26 時点の詳細は未確認。
- GitHub MCP Server Secret Scanning: 個人開発でも API キー流出防止に即導入価値あり。課金: GitHub Secret Protection / Advanced Security 条件はリポジトリ種別で要確認。
- Anthropic Stainless / MCP server tooling: 自作 SaaS の API を SDK / MCP 化し、Claude Code や Codex から操作可能にする導線として有望。課金・個人利用条件は未確認。

## Claude Code/Codex連携可能性
- GitHub MCP Server: Claude Code / Codex / Copilot 系から Issue、PR、secret scanning を呼び出し、実装前後の安全確認に使う。
- Anthropic Remote MCP Connectors: Asana、Atlassian、Cloudflare などを Claude に接続し、個人開発のタスク管理・デプロイ補助に使う。
- Stainless: OpenAPI 仕様から SDK / CLI / MCP server を生成し、自作サービスを Agent 操作用 API として公開する。
- Google ADK / Gemini API Managed Agents: Codex で Agent コードを生成し、Gemini 側で永続実行環境つき Agent として動かす構成が考えられる。
- Codex Skills / Automations: Figma、Vercel、画像生成などの skill を組み合わせ、UI 実装からデプロイまでの反復作業を半自動化する。

## MCP/Agent系情報
- [GitHub MCP Server secret scanning GA](https://github.blog/changelog/2026-05-05-secret-scanning-with-github-mcp-server-is-now-generally-available/): MCP 対応 IDE / Agent から漏洩シークレット検出が可能。個人開発でも最優先で試す価値あり。
- [GitHub Copilot cloud agent configuration REST API](https://github.blog/changelog/2026-05-18-audit-repository-copilot-cloud-agent-configuration-via-the-rest-api): MCP server 構成、許可ツール、Actions policy、firewall 設定を監査可能。個人よりチーム向け。
- [Anthropic acquires Stainless](https://www.anthropic.com/news/anthropic-acquires-stainless): SDK / MCP server 生成領域を Anthropic が取り込み。API を持つ個人開発サービスの Agent 対応が進みそう。
- [modelcontextprotocol 公式 GitHub](https://github.com/modelcontextprotocol): Python / Go / Rust / Java / C# SDK などが継続更新。2026-05 時点で MCP 実装の標準候補。
