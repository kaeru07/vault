## 個人開発で活用できそうなもの
- GitHub Copilot app・CLI: 複数 issue、PR、検証、定期メンテを agent に分担させ、個人開発を小規模チームのように回せる。課金: Copilot Pro $10/月、Pro+ $39/月、無料枠あり。
- Buzzy Builder MCP: 管理画面、簡易SaaS、モバイル風MVPを Figma/no-code/MCP 経由で短期生成する用途に向く。デプロイ課金 $20/月から、BYO OpenAI API key 可。
- Kurrent Capacitor: Codex・Claude Code・Cursor を併用する場合、過去の調査・判断・実装理由を再利用できそう。private preview のため即導入性は低い。
- NetFoundry MCP Gateway: VPS 上の自作 MCP を外部公開せず agent へ接続する設計の参考になる。個人向け価格は未確認。
- Microsoft Web IQ・Foundry hosted agents: 市場調査 bot、競合監視、レポート生成 agent の grounding と評価に使えそう。ただし価格・個人利用条件は未確認。

## Claude Code/Codex連携可能性
- Buzzy Builder MCP: Codex・Claude Code から MCP 経由でアプリ定義を作り、Buzzy 側で web/native app 化する。
- GitHub Copilot app MCP: GitHub issue・PR・review を Copilot 側で管理し、Codex/Claude Code は実装・検証担当として分業する。
- Kurrent Capacitor: 複数 agent の過去セッションを検索し、Codex の再実行や Claude Code の設計レビューに渡す。
- NetFoundry MCP Gateway: ローカル・社内 MCP サーバーを安全に Codex/Claude Code へ接続する。
- Microsoft Foundry Agent Service: Codex で作った agent workflow を Azure 側の hosted agents、ASSERT、Agent Control Specification で運用・評価する。
