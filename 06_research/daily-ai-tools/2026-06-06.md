## 個人開発で活用できそうなもの
- GitHub Copilot app・CLI: 複数 issue、PR、検証作業をエージェントに分担させ、個人開発でも小規模チーム的に開発を回せる。課金: Copilot Pro $10/月、Pro+ $39/月、無料枠あり。
- Buzzy Builder MCP: Figma・no-code・MCP を使い、管理画面や簡易モバイルアプリの MVP を短期生成する用途に向く。デプロイ課金 $20/月から、AI credits は従量または BYO key。
- NetFoundry MCP Gateway: 自宅サーバーや VPS 上の MCP ツールを公開せず Codex/Claude Code へ渡す構成に応用できそう。ただし個人利用価格は未確認。
- Kurrent Capacitor: 複数コーディングエージェントを使う場合、作業履歴・判断理由の再利用に有用そう。private preview のため即導入性は低い。
- Microsoft Web IQ: 個人開発の調査エージェントや市場調査 bot の grounding に使えそうだが、Foundry 側の価格・利用条件は未確認。

## Claude Code/Codex連携可能性
- Buzzy Builder MCP: Codex・Claude Code から MCP 経由でアプリ定義を生成し、Buzzy の実行基盤で web/native app 化する。
- GitHub Copilot SDK: Codex/Claude Code で作った内部ツールを Copilot runtime 側へ寄せ、GitHub issue・PR・review と接続する。
- Workday Agent-Ready Tools: Claude Code・Codex などの agentic tool から Workday データへ MCP 経由で制御アクセスする企業向け連携。
- NetFoundry MCP Gateway: ローカル・社内 MCP サーバーを外部公開せず、Codex/Claude Code などの MCP client へ安全に接続する。
- Microsoft Agent Framework・Foundry Agent Service: MCP、A2A、Responses API、Playwright CLI 等を組み合わせ、Codex で作った agent workflow を Azure 側で運用する。
