## 個人開発で活用できそうなもの
- Google ADK 2.0: 無料OSSなので、調査エージェント、SNS投稿生成、アプリ競合分析などの自動化を低コストで試せる。セルフホスト可。
- OpenAI Codex Record & Replay: 手作業で行うリリース前チェック、スクレイピング手順、UI検証手順をSkill化できる可能性がある。課金: Plus $20/月以上。
- Cloudflare Agents MCP: 個人開発サービスにMCP APIを付け、AIツールから自サービスを操作できる導線を作れる。Cloudflare依存、価格詳細は未確認。
- Claude Code MCP: GitHub、Sentry、DB、Figmaなどをつなぎ、実装から調査、運用確認まで1つの開発Agentに寄せられる。API利用時は従量課金。
- Arcade.dev: 将来、個人開発SaaSで「AIに操作させるが権限は絞る」実装に使える可能性。価格・個人利用可否は未確認。

## Claude Code/Codex連携可能性
- Claude Code + MCP: `claude mcp add` でHTTP、stdio、WebSocketのMCP serverを追加し、issue、DB、監視、メール下書きなどを直接操作。
- Codex + MCP: Codex docsにMCP、Codex SDK、MCP Server導線があり、開発Agentから外部ツールを呼ぶ構成に使える。詳細設定は要再確認。
- Google ADK + Codex: ADKのWorkflow/Task APIをCodexで生成・修正し、個人用Agent基盤を反復開発。
- Cloudflare Agents + Claude Code: Cloudflare上にRemote MCP serverを置き、Claude Codeから自作サービスの管理・分析ツールとして接続。
- GitHub MCP Registry + Claude/Codex: 既存MCP serverを探して、GitHub、Postgres、Slack等の連携を短時間で試す。
