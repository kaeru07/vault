## 個人開発で活用できそうなもの

- GitHub Copilot App: 個人開発の複数Issueを並列で進め、MCPでNotion/GitHub/ローカルツールをつないでレビューまで回す用途。既存 Copilot 課金があるなら導入価値が高い。
- GitHub Copilot CLI: `/after` や `/every` で定期メンテ、README更新、軽い調査をCLIから予約できる可能性がある。個人VPSの運用タスクと相性がよい。
- Google Antigravity 2.0 / Managed Agents: AndroidアプリやFirebase系の短期プロトタイプ作成に向く。課金: AI Ultra $100/月が明示されており、個人にはやや高め。
- MWM AI Mobile Squad: App Store / Google Play 公開、IAP、成長施策まで含むなら個人開発の収益化導線に近い。ただし価格・一般利用可否は未確認。
- Claude remote MCP Custom Connectors: 自作MCPをClaudeに接続して、Vault、GitHub、分析DBなどを安全に呼び出す実験に使える。無料枠は1 connectorまで。
