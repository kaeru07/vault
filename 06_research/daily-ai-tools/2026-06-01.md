## 個人開発で活用できそうなもの
- Google Antigravity 2.0: Android/Firebase/AI Studio連携が強く、個人開発のプロトタイプからテストトラック公開まで短縮できる可能性が高い。
- Managed Agents in the Gemini API: SaaS内に「調査・コード実行・ファイル処理」エージェントを組み込む用途に向く。API単価は未確認。
- OpenAI Codex Goal mode: 既存リポジトリの長時間タスク、UI修正、検証ループを任せやすく、個人開発の保守作業に直結。
- Vercel AI Gateway: 複数モデルの切替・許可リスト管理ができ、AI機能付きWebアプリの運用コスト管理に使える。
- Claude Code dynamic workflows: 大きめのリファクタ、複数ファイルの実装、MCP連携前提の開発に向く。

## Claude Code/Codex連携可能性
- Vercel AI Gateway: Codex/Claude CodeでAI SDK実装を生成し、Gateway経由でモデル切替・利用制限を管理。
- GitHub Copilot app / Copilot CLI: GitHub issueやPRを起点にAgent作業を開始し、Codexのレビューと組み合わせる。
- Google Antigravity SDK: Codexで生成したアプリに、Gemini系Agent harnessをAPI連携する構成が可能。
- OpenAI Agents SDK: CodexでAgent実装を進め、サンドボックス・ファイルツール・長時間実行基盤を組み込む。
- Claude Code dynamic workflows: MCPサーバーと組み合わせ、設計・実装・検証を長めの作業単位で並列化。

## MCP/Agent系情報
- GitHub Copilot CLI: GitHub MCP server内蔵、custom MCP server対応。CLI上のAgent開発基盤として実用度が高い。公式: https://github.blog/changelog/2026-02-25-github-copilot-cli-is-now-generally-available/
- Google Antigravity 2.0 / Managed Agents: Antigravity agent harness、永続Linux環境、custom instruction/skillsに対応。公式: https://blog.google/innovation-and-ai/technology/developers-tools/google-io-2026-developer-highlights/
- Vercel MCP / AI SDK MCP client: Vercel上でMCP serverをデプロイし、AI SDKからMCP client接続可能。公式: https://vercel.com/docs/mcp
- Cursor MCP Apps: Agent chat内でAmplitude/Figma/tldraw等の対話UIを表示。2026-05時点の最新性は未確認だが、MCPのUI拡張例として重要。公式: https://cursor.com/en/changelog
