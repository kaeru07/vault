## 個人開発で活用できそうなもの
- GitHub Copilot desktop app: 複数リポジトリやIssue単位でAIエージェントを走らせる用途に向く。個人開発でも「バグ修正」「LP生成」「テスト追加」を並列化できる可能性がある。
- OpenAI Codex: 既存リポジトリの調査、修正、テスト実行、PR単位の作業に向く。課金プラン前提だが、個人開発の実装量産には最も直接的。
- Arcade.dev: OAuth・SaaS操作・社内ツール連携をAgentに任せる場合の認可レイヤーとして有望。個人開発ではMCPサーバー公開時の権限制御に使える可能性。
- OpenClaw: セルフホスト型の個人AI秘書・自動化基盤として有望。ただし認証情報やローカルファイルへのアクセスリスクが高く、本番端末では非推奨。
- Project Rayfin: 個人開発で「AIが作った試作品をそのまま管理バックエンドへ接続する」用途に合いそうだが、現時点では価格・一般提供状況が未確認。
