## 個人開発で活用できそうなもの
- Microsoft Intelligent Terminal: Windows個人開発者なら、エラー調査、コマンド生成、ローカルのCodex/Claude Code起動導線をターミナルに集約できる。セルフホストではなくローカルアプリ型。
- ADK Arena: 自作AIエージェント基盤を選ぶ前に、フレームワークの使いやすさ・失敗率・実装コストを比較する評価軸として使える。
- Project Rayfin: 個人開発MVPをバックエンド込みで短期公開する用途に合う可能性があるが、Microsoft Fabric依存と価格未確認がリスク。
- MCP Apps: チャット内に管理画面、フォーム、可視化UIを出せるため、単なるMCPツールより有料SaaS化しやすい。
- GitHub Copilot desktop app: 複数Issueや小修正を並列で進める運用に向くが、クレジット消費監視が必須。

## Claude Code/Codex連携可能性
- Microsoft Intelligent Terminal: ACP対応によりClaude CodeやOpenAI Codexをターミナルのエージェントペインから切り替えて使える可能性がある。
- MCP Apps: Codex/Claude CodeからMCPサーバーを経由し、ツール実行だけでなくレビューUIや承認フォームを返す構成にできる。
- GitHub Copilot desktop app: GitHub Issue/PR起点のタスクを他エージェントと比較しながら進める司令塔として使える可能性がある。
- ADK Arena: Codexに複数ADKで同じエージェント実装を生成させ、成功率・テスト通過率で採用技術を選べる。
- Ona系クラウド実行環境: Codexの長時間タスク、永続セッション、非同期実行の基盤として統合される可能性があるが、買収完了と提供形態は未確認。
