## 個人開発で活用できそうなもの
- AWS MCP Server: 個人開発のAWS構成作成、CloudWatch調査、Lambda/S3/DynamoDBの試作をCodexやClaude Codeから半自動化できる。追加料金なしだが、AWSリソース課金に注意。
- Google Antigravity 2.0: FirebaseやAndroidを使う小規模アプリの初期実装を並列エージェントで進められる可能性が高い。課金 $100/月のため、短期集中開発向け。
- OpenAI Codex Appshots + Goal mode: UIのスクリーンショット文脈をCodexへ渡し、フロントエンド修正や検証の反復を短縮できる。
- GitHub Copilot CLI agent in JetBrains: JetBrains系IDE利用者ならworktree隔離で既存作業を壊さずに修正案を並列生成できる。
- Gemini API Managed Agents: SaaS内の自動実行エージェント機能をAPIで組み込む候補。ただし料金・制限の詳細は未確認。

## Claude Code/Codex連携可能性
- AWS MCP Server: Claude Code/CodexからAWS APIを安全に呼び、インフラ調査・デプロイ補助・ログ確認を自動化。
- Salesforce Data 360 MCP Server: Claude CodeやCursorからData 360の顧客データ文脈を参照し、CRM連動アプリの実装補助に利用。
- OpenAI Codex Appshots: Codexにアプリ画面を添付し、UI差分修正・CSS調整・受け入れ条件チェックを依頼。
- GitHub Copilot CLI agent: JetBrains上でCLI agentを呼び、worktree単位の修正案を生成してレビュー。
- Gemini API Managed Agents: Codexで書いたアプリからGeminiのManaged Agentを呼び、バックグラウンド実行タスクを外部化。
