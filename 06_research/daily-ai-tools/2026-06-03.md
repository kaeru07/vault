## 個人開発で活用できそうなもの
- Codex Goal mode: 小規模SaaSや学習アプリの「完了条件付き」実装タスクに向く。UI修正、テスト追加、既存PR更新の量産に使える。
- GitHub Copilot + MCP Server secret scanning: 個人開発でもAPIキー漏洩の予防に直結。GitHub連携前提だが、収益化前の信用毀損リスクを下げられる。
- Claude Design: LP、アプリ画面、Pitch資料の初稿生成に使える。動画化・SNS投稿用の視覚素材作成にも展開しやすい。
- Base MCP: 暗号資産・オンチェーン決済系の個人プロダクト検証には有用。ただし金融・秘密鍵・規制リスクが高く、実運用は慎重に分離環境で試すべき。
- GitHub Copilot VS Code agent改善: 既存リポジトリの横断検索、履歴参照、エージェント作業監視が強化され、個人開発の保守タスク短縮に向く。

## Claude Code/Codex連携可能性
- GitHub MCP Server: Claude Code/Codexからリポジトリ操作、secret scanning、PR前検査を組み込める。
- Base MCP: Claude、ChatGPT、Cursor、Codex等との接続可能性が報告されているが、Codexでの公式接続手順は未確認。
