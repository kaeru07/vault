## 個人開発で活用できそうなもの
- GitHub Copilot app: Issue整理、依存更新、リリースノート作成、軽微PR作成を定期タスク化できる。利用量課金のため上限管理が前提。
- MCP Registry: 自作MCPサーバーを公開・検証する導線として有用。セルフホスト可なので個人開発の配布基盤にも向く。
- OpenAI Codex: コード以外にスプレッドシート処理、調査、ダッシュボード生成へ広げられる。ChatGPT Plus/Pro/Team系の課金前提。
- Gemini Spark: まだTrusted tester/Beta段階だが、通知・メール・小タスク自動化の個人エージェント化に近い。日本提供は未確認。
- Microsoft Web IQ: 直接利用条件は未確認だが、MCP-native検索基盤として調査Bot・市場監視Botへの応用余地あり。
