## 個人開発で活用できそうなもの
- AWS MCP Server: AWS上の個人SaaSや自動化基盤をCodex/Claude Codeから安全に操作する候補。無料追加料金なしだが、AWS実リソース課金には注意。
- Google Antigravity 2.0 / Managed Agents: Androidアプリ、Firebase連携、プロトタイプ量産に向く。Google AI Ultra $100/月は個人には重めなので、無料枠・低額枠は未確認。
- GitHub Copilot cloud agent REST API: Issue起点で修正PR、週次リリースノート、複数リポジトリの一括更新を自動化できる。Business/Enterprise中心で個人Pro対応は今後予定。
- Circle Agent Stack: AIエージェント課金、マイクロペイメント、ツール利用ごとのUSDC決済の実験に使えそう。日本の個人開発での規制・会計面は未確認。
- Honeycomb Agent Observability: LangGraph/Codex/Claude Code系の本番エージェント失敗分析に有用。個人利用価格は未確認。

## Claude Code/Codex連携可能性
- AWS MCP Server: Claude Code/CodexからMCP経由でAWS APIを呼び、CloudTrail付きでインフラ操作を監査する。
- GitHub Copilot cloud agent REST API: CodexがIssue整理、Copilot agentがPR作成、Codexがレビューという分担が可能。
- UiPath for Coding Agents: Claude CodeやCodexが生成した自動化をUiPathの統制・監査・実行基盤へ載せる連携候補。
- Circle Agent Stack: Codex製のエージェントにAgent WalletやNanopaymentsを組み込み、ツール利用課金を試す。
- Gemini API Managed Agents: Codexで設計したタスクをGemini側の隔離実行エージェントへ委譲する構成が考えられる。
