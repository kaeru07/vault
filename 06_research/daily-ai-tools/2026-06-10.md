## 個人開発で活用できそうなもの
- OpenAI Remote MCP: 自作SaaSの管理画面、Stripe、Google Calendar、DB参照をAgentから安全に呼ぶ収益化アプリに向く。承認必須設計にすれば個人開発でも事故を減らせる。
- Claude Code Agent SDK: 定型のPRレビュー、依存更新、翻訳、テスト生成を自作Agent化できる。小規模な開発代行・保守自動化サービスに使いやすい。
- GitHub Copilot Cloud Agent: GitHub Issueを起点に修正PRを作る運用と相性が良い。個人開発の複数プロダクト保守を省力化できる。
- MCP Apps: Figma・Canva・SlackなどのUIをチャット内で扱う流れは、ノーコード寄りの制作支援ツールや動画素材生成に転用余地あり。
- Model Context Protocol: 自作MCPサーバーを作れば、既存データや社内ツールをClaude/Codex/VS Code/Cursorに横展開できる。セルフホスト可否: 実装次第で可。

- Claude Code MCP: Claude CodeからGoogle Drive、Jira、Slack、自作ツールを参照し、仕様確認から実装・PR作成までつなげる。
