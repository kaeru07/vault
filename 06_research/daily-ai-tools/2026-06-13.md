## 個人開発で活用できそうなもの
- OpenAI Codex SDK: 個人開発の定型作業、Issue分解、テスト修正、リリース前レビューを自動化しやすい。課金は従量・credits消費なので上限管理が必須。
- GitHub Copilot app / Cloud agent: GitHub Issuesから小さな修正PRを作らせる用途に向く。無料枠ありだが、本格利用はPro 10$/月以上。
- GitHub MCP Registry: Notion、DB、デプロイ先、課金基盤などをMCPで統合できれば、個人開発の運用自動化に直結する。
- Claude Code: 既存コードの読解・修正・テスト生成に強く、月20$から試せる。Claude Proに含まれる点は個人開発向き。
- Cloudflare Agents MCP: Cloudflare Workers系で低コストなAIエージェントAPIを作る場合に相性がよい。セルフホストというよりCloudflare上での運用。

## Claude Code/Codex連携可能性
- MCP: Claude Code / Codex双方の外部ツール接続基盤として使える。DB、GitHub、ブラウザ、決済管理などを安全に小分け接続する。
- AGENTS.md / repository instructions: Codexや他の coding agent に実装方針・禁止事項・テスト方針を共有できる。
- GitHub Copilot third-party agents: GitHub上でClaude CodeやCodexを選択してIssue対応やPRレビューに使える可能性がある。公式: https://github.com/features/copilot/plans
- Codex SDK: CodexをCLI/CI/スクリプトに組み込み、日次棚卸し、差分調査、修正PR作成に連携できる。
- Cloudflare Agents + MCP: 自作MCPサーバーをWorkers上でホストし、Claude Code/Codexから呼び出す構成が現実的。
