## 個人開発で活用できそうなもの
- OpenClaw: セルフホスト可能な個人アシスタント基盤として、通知、調査、定期実行、簡易ワークフロー自動化に使いやすい。
- Antigravity 2.0: Gemini系のCLI・SDK・プラグインが揃えば、個人開発の実装、調査、テスト生成を1環境に寄せられる可能性がある。
- MCP Apps / MCPサーバー: 個人開発SaaSに「AIから操作できるUI・API」を提供する導線として有望。ただし権限設計が必須。
- GitHub Copilot desktop app: 複数リポジトリや小規模アプリを並列で進める用途に合うが、従量課金化でコスト監視が必要。
- IBM Bob: 個人開発では重いが、レガシーコード解析・セキュリティ観点のベンチマーク対象としては参考になる。

## Claude Code/Codex連携可能性
- MCPサーバー: Claude Code/CodexからGitHub、DB、ブラウザ、決済、分析基盤を呼び出す共通接続層として使える。
- OpenClaw: ローカル常駐エージェントにCodex/Claude Codeで生成したスクリプトやMCP接続を組み込める可能性。
- Antigravity 2.0 SDK: Gemini系エージェントのワークフローをClaude Code/Codexで生成・検証する補助線になる。詳細APIは未確認。
- GitHub Copilot desktop app: GitHub中心のタスク管理とCodexのローカル実装を分担する運用が考えられる。
- MCP Apps: CodexでMCPサーバーとReact UIを実装し、Claude/ChatGPT側から操作可能なミニ管理画面にできる可能性。

- MCPセキュリティ: STDIO処理、ツール権限、プロンプトインジェクション、サプライチェーン汚染のリスク報告あり。個人開発でもallowlistとサンドボックス必須。
- AIP: MCP/A2Aをまたぐエージェント本人性・委任検証プロトコル案。URL: https://arxiv.org/abs/2603.24775
- WorkBench Revisited: 2026年時点の職場エージェント性能改善を扱う研究。実用化は進むが、誤送信など不可逆ミスは残る。URL: https://arxiv.org/abs/2606.13715
