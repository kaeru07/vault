## 個人開発で活用できそうなもの
- ADK for Android: 学習アプリ、家計簿、旅行メモなどで個人データを端末内処理しつつ、必要時だけクラウドAIへ委譲する設計に向く。
- Genkit Middleware: 小規模SaaSのAI機能に承認フロー、失敗時fallback、ログを最初から入れられるため、課金ユーザー向けの品質担保に使いやすい。
- Google AI Edge Gallery MCP対応: スマホ常駐の個人用自動化、リマインダー、ローカル文書検索のプロトタイプに向く。
- Codex mobile preview: 外出中に個人開発のPRレビュー、テスト承認、方向修正ができ、短期リリースの待ち時間を減らせる。
- WebMCP: まだ提案段階だが、WebアプリをAIエージェント対応にする標準候補として早めに観察価値あり。

## Claude Code/Codex連携可能性
- ADK for Kotlin / Android: Claude Code や Codex で Kotlin エージェント雛形、MCP tool、Android デモを量産する。
- Genkit Middleware: Codex に middleware test、human approval hook、observability hook を実装させる。
- MongoDB MCP Server: Claude Code / Open Code 互換が明記されており、個人SaaSのDB調査・インデックス確認に接続可能。
- AWS MCP Server: Claude Code / Codex 系の coding agent から AWS API を監査付きで呼ぶ基盤として使える。追加料金なし、AWS利用料は別途。
- Codex mobile: Codex 実行中の承認・レビュー・指示変更をスマホから行い、VPSやMac上の作業を継続する。
