## 個人開発で活用できそうなもの
- Open Agents: 個人開発の「PR作成エージェント」「バグ修正エージェント」をVercelテンプレートから短期構築できる。GitHub連携込みなのでSaaS化しやすい。
- GitHub Copilot Sandboxes: AIにローカルコードを触らせる時の事故範囲を狭められる。副業・個人開発で本番秘密情報を守る実行基盤として有用。
- Antigravity CLI: Gemini CLI利用者は2026-06-18までに移行検証が必要。無料枠終了リスクがあるため、Codex/Claude Codeとの併用前提で比較したい。
- Copilot CLI prompt scheduling: 毎朝の市場調査、issue棚卸し、リリース前チェックをCLIから定期実行できる。小規模自動化の収益化に向く。
- Claude Opus 4.8: 長時間の設計・リファクタ・コードレビューで使い道がある。課金コストが高い場合は最終レビュー専用に絞るのが現実的。

## Claude Code/Codex連携可能性
- Open Agents: Codex/Claude Codeで生成した修正方針を、Vercel SandboxとWorkflowに渡してバックグラウンドPR化する。
- GitHub Copilot App MCP: 既存MCPサーバーを接続し、CodexやClaude Codeで作った内部ツールをCopilot側のagent surfaceから呼ぶ。
- MoEngage Merlin AI Custom Agents: MCP経由で外部CRM・分析基盤と接続できるため、Codexで作った小規模マーケ自動化APIを呼び出す余地あり。
- Antigravity CLI: Claude Code/Codexの比較対象として、同一repo・同一MCPサーバーで実装品質とコストを測る。
- GitHub Copilot CLI scheduled prompts: Codexで作った調査・テストスクリプトを、Copilot CLIの定期プロンプトから実行する構成が考えられる。
