## 個人開発で活用できそうなもの
- GitHub Copilot app: Issue 駆動で修正案・PR作成を回す用途。個人開発では小規模SaaSのバグ修正、README整備、テスト追加の自動化に向く。
- Project Rayfin: バックエンド構築を短縮できるなら、検証用アプリや広告収益アプリの MVP 化に使える可能性がある。ただし Microsoft Fabric 依存と価格は未確認。
- Gemini Spark: Workspace、メール、スプレッドシート、将来 MCP 連携を使った日次調査・競合監視エージェントに向く。個人利用は beta/Ultra 前提のため費用対効果は未確認。
- Arcade.dev: MCP ツールを外部 API に接続する個人開発では、権限スコープと監査の実装参考になる。直接導入より設計パターンとして価値が高い。
- IBM Bob: COBOL/RPG 等のレガシー対応が強みなので、個人開発一般より受託・業務改善系に寄る。課金は最低 $20/月。

## MCP/Agent系情報
- MCP の利用拡大: 2026年初頭時点で MCP サーバー・SDK ダウンロードが大きく伸び、研究では 177,436 agent tools の分析が報告された。根拠: https://arxiv.org/abs/2603.23802
- MCP セキュリティ: stdio handling や tool poisoning、権限境界の弱さが継続課題。Tom's Hardware は公式 SDK 群の RCE リスク報道を掲載。根拠: https://www.tomshardware.com/tech-industry/artificial-intelligence/anthropics-model-context-protocol-has-critical-security-flaw-exposed
- A2A + MCP 認可: Agent Identity Protocol など、MCP/A2A を跨ぐ delegation・capability token の研究が出ている。根拠: https://arxiv.org/abs/2603.24775
- MoSPI MCP server: インド統計省が政府統計データ向け MCP server beta を公開したと報道。公的データを AI ツールに直結する事例。根拠: https://m.economictimes.com/news/india/mospi-launches-mcp-server-to-link-ai-tools-with-govt-data/articleshow/128005462.cms
