## 個人開発で活用できそうなもの
- GitHub Spark: 小規模SaaS、学習アプリ、管理画面のMVP生成に向く。課金前提だが、短期リリース目的なら試す価値が高い。
- Intelligent Terminal: Windows環境で個人開発している場合、CLIエラー対応や環境構築の詰まりを減らせる可能性がある。
- Windows Developer Configurations: VPS・検証PC・配信環境の再現性を上げられる。AI工場的に複数プロジェクトを量産する場合に実用的。
- Microsoft Agent Framework 1.0: Python/.NETで、調査、要約、実装、レビューを分担する自前エージェントを作る用途に合う。
- MCP Apps: 個人開発者が「AIチャット内で操作できる小型UI」を提供する入口になり得る。MCPサーバー実装が必要。

## Claude Code/Codex連携可能性
- MCP Apps: Claude Code/CodexからMCPサーバーを呼び、生成結果をテキストだけでなく小型UIで確認する連携が考えられる。
- Microsoft Agent Framework 1.0: CodexでSDK実装を生成し、複数エージェントのタスク分解・検証パイプラインを組む。
- GitHub Copilot coding agent: CodexやClaude Codeで作ったIssue仕様をCopilot coding agentへ渡し、PR作成まで非同期化する。
- ADK Arena: Claude Code/CodexでAgent SDK候補を実装比較し、個人開発のエージェント基盤選定に使う。
- Windows Developer Configurations: Codexで`dev-config.winget`を生成・レビューし、新規開発端末の初期化を自動化する。
