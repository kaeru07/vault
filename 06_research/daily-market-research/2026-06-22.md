# 市場調査ログ 2026-06-22

## 今日の結論
次に見るべきは「AI エージェントの権限管理」と「縦型マイクロドラマの課金導線」を個人開発サイズへ落とす方法。  
AI アシスタント市場は ChatGPT 一強から Gemini / Claude / Meta AI へ分散し、比較・乗り換え支援ツールに余地がある。  
MCP / A2A / agent authorization は企業向けに見えるが、個人開発では「安全なローカル自動化」「操作ログ監査」の小型 SaaS 化が狙える。  
短尺動画領域は AI 生成コンテンツとアプリ内課金の接点が強く、動画化・広告・テンプレ販売と相性がよい。

## 競合アプリ
- ChatGPT / OpenAI / 汎用 AI アシスタント、Web・モバイル・音声・画像生成 / 強み: 最大級の認知と利用規模 / 弱み: 価格・利用制限・信頼面の不満が出やすい
- Gemini / Google / Android、Search、Workspace との統合 / 強み: Google エコシステム流入 / 弱み: 単体アプリとしての差別化は未確認
- Claude / Anthropic / 長文・開発・エージェント利用に強い AI アシスタント / 強み: 開発者・業務利用での評価 / 弱み: 高額プラン上限の透明性に不満報道あり
- ReelShort / Crazy Maple Studio / 縦型マイクロドラマ配信 / 強み: 課金導線が明確で米国市場に強い / 弱み: 広告投資依存・制作費増の可能性
- My Drama / HOLYWATER TECH / 短尺縦型ドラマ、AI 支援コンテンツ展開 / 強み: 多言語・スマホ特化 / 弱み: 競合が多く差別化が難しい

## 収益化ヒント
- AI サブスク比較ダッシュボード: ChatGPT / Gemini / Claude / Perplexity の料金、制限、用途別おすすめを更新し、アフィリエイト・note・広告へ接続。

- AI 利用上限トラッカー: 各 AI サービスの利用回数・課金額・作業ログを手入力または API 連携で管理。個人開発者向け SaaS。
