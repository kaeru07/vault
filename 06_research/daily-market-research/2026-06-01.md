# 市場調査ログ 2026-06-01

## 今日の結論
次に見るべきは「AIアシスタント本体」ではなく、AI動画・AIエージェント・AIノートなど高付加価値の垂直領域である。  
App Store / Google Play の収益上位では ChatGPT、Claude、Grok、Perplexity が強く、汎用AIは大手寡占が進んでいる。  
一方で Sensor Tower 系の情報では、AIコンパニオン、AIエージェント、AI画像・動画生成はまだ中小プレイヤーの余地がある。  
個人開発では「既存AIを使う小さな業務特化アプリ」と「Play Shorts / Shorts向けの動画化しやすい導線」を優先するべき。

## 注目ジャンル
- AI動画生成 / Google Flow、YouTube Shorts、Gemini Omni / 短尺動画・広告素材・解説動画に直結し、動画化と課金導線を作りやすい
- AIエージェント / Manus、Gemini Spark、Poke系 / 汎用チャットより「作業完了」に近く、月額課金の理由を作りやすい
- AIノート・文字起こし / Plaud、Otter、Summary、Speechify / 会議・学習・営業など用途が明確で継続課金に向く
- AIコンパニオン / Character.ai、Polybuzz系 / 感情的ニーズが課金に転換しやすいが、品質・安全性・差別化が課題
- App Store / Google Play向けASO支援 / App Vulture、AppBrain、Google Play Console / AIアプリ増加で発見性が悪化し、開発者向けツール需要がある
- ショート動画発見導線 / Google Play Shorts、YouTube Shorts / アプリ紹介そのものが動画化され、個人開発アプリも動画素材が必要になる

## 競合アプリ
- ChatGPT / OpenAI / 汎用AIアシスタント、モバイルIAP収益が強い / 強み: ブランド・利用習慣・機能拡張 / 弱み: 個別業務への深い最適化は外部アプリに余地
- Claude by Anthropic / Anthropic / 推論・コーディング寄りのAIチャット / 強み: 開発者・知的作業層への訴求 / 弱み: 消費者向け動画・遊び用途は相対的に弱い
- Grok - AI Chat & Video / xAI / AIチャットと動画・リアルタイム文脈 / 強み: X連携と話題性 / 弱み: 業務特化・日本語ローカル用途は未確認
- Perplexity / Perplexity AI / AI検索・回答エンジン / 強み: 調査用途の明確さ / 弱み: ワークフロー完結型ではない
