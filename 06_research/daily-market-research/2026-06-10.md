# 市場調査ログ 2026-06-10

## 今日の結論
次に見るべきは「AIアプリの巨大化で空く周辺市場」、特に ChatGPT / Gemini / Claude 本体では拾いきれない職種特化・短尺動画化・業務連携の隙間。  
ChatGPT が月間10億ユーザー規模に到達したという二次情報があり、汎用AI本体との正面競争は避けるべき。  
一方で、Google Play のAI発見導線、TikTok/CapCut系のAI動画広告、MCP/Agent系の実務連携は個人開発でも小さく切り出せる。  
短期収益化では「AIで作る」より「AI成果物を売る・配る・測る」補助ツールが現実的。

## 注目ジャンル
- AIアプリ発見・ASO支援 / Google Play が Ask Play、Play Shorts、Gemini連携を進めており、検索キーワード以外の発見導線が重要化 / Google Play、Gemini
- AIチャット周辺ツール / ChatGPT の月間10億ユーザー到達という二次情報があり、利用者母数の周辺需要が大きい / App Store、Google Play、AIニュース
- AI短尺動画広告 / CapCut、Pippit、Creatify、TikTok Smart Split 系の流れで、商品URLから動画広告を作る需要が継続 / TikTok、CapCut、Instagram Reels
- エージェント用メール・業務連携 / Hostinger Agentic Mail のように、人間向けメールをAIワークフロー向けに再設計する動き / AI Agent、MCP、n8n
- モバイルAIエージェント評価 / iOSWorld のようなスマホ操作エージェント評価が出ており、個人向け自動操作アプリの期待と限界が見える / AI研究、iOS、Agent
- クリエイター収益支援 / TikTok Shop affiliate、Creator’s Corner買収報道など、教育・コミュニティ・テンプレ販売が収益化導線になりやすい / TikTok、SNS、note

## 競合アプリ
- ChatGPT / OpenAI / 汎用AIチャット、画像生成、業務支援の中心アプリ / 想定される強み: 圧倒的な利用者母数とブランド / 弱み: 個別業務の深いUIや日本語ローカル運用は外部余地あり
- Claude / Anthropic / 文章・開発・業務利用に強いAIチャット / 想定される強み: 長文処理、開発者・知識労働者の支持 / 弱み: モバイルの動画・診断・広告収益体験とは距離がある
- Gemini / Google / Google Play、Search、Workspace、Androidと結びつくAI / 想定される強み: OS・検索・アプリ発見導線との統合 / 弱み: 個人開発者には仕様変更依存リスクがある
- CapCut / ByteDance / AI字幕、テンプレ、動画生成、広告素材作成を含む動画編集アプリ / 想定される強み: TikTok連携と制作テンプレの強さ / 弱み: 高機能化で初心者や特定職種には過剰な可能性
- Pippit / ByteDance / 中小企業・クリエイター向けAI動画・画像制作プラットフォーム / 想定される強み: 商品販促動画への明確な用途 / 弱み: 詳細な価格・継続利用満足度は未確認

## ユーザー不満
- AI動画ツールは便利でも、細かい編集制御やブランドトーン調整が難しいという不満があり得る。情報源は二次情報と研究要旨ベースで未確認。
- AIチャット本体は強いが、成果物を投稿用、営業用、教材用に整える作業が残る。実ユーザー投稿での確認は未確認。
- Google Play / App Store のランキングや広告効果は外部から追いにくく、個人開発者が日次で判断しにくい。前日ログとも重複。
- AIエージェントは複数アプリ横断操作がまだ難しい。iOSWorld 要旨では最良構成でも全体52%、複数アプリタスク37%とされるが、実利用環境での再現性は未確認。
- TikTok Shop やAI量産コンテンツは、低品質商品・広告過多・収益悪化への不満が出やすい。個別SNS投稿の本日確認は未確認。

### 収益化との関係
本日の内容は個人開発の収益化に近い。

ただし、ランキングやSNS不満の一次確認が不足しており、実需要の強さは未確認部分が残る。
