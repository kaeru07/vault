# 市場調査ログ 2026-06-12

## 今日の結論
次に見るべきは「AI検索・AI発見経由でアプリが見つかる導線」と、短尺動画でCVRを上げるGoogle Play / YouTube型の配信面である。  
Google PlayはAsk Play / Play Shortsで、アプリ発見がキーワード検索から会話検索・動画プレビューへ寄っている。  
AIアプリ市場はChatGPT / Claude / Gemini / Meta AIなど大手が上位を占め、個人開発は汎用AIではなく用途特化・ワークフロー特化が現実的。  
Codex / Claude Code系の普及により、短期試作できる小型SaaS・業務自動化・学習支援アプリの量産余地が増えている。

## 注目ジャンル
- AIアプリ発見最適化 / Ask PlayやGemini経由で発見される可能性 / Google Play

- ショートドラマ・縦型動画課金 / ReelShort、DramaBox系のイベント露出とアプリ内課金 / Google Play・SNS

## 競合アプリ
- ChatGPT / OpenAI / 汎用AIチャット・作業支援 / 強み: ブランドと利用頻度 / 弱み: 個別業務への深い最適化は利用者側の設定依存
- Claude / Anthropic / コーディング・文書・業務支援 / 強み: Claude Codeとの連携、長文・開発用途 / 弱み: 高度機能は課金前提になりやすい
- Google Gemini / Google / Android・検索・Play連動のAI体験 / 強み: OS・検索・Playへの統合 / 弱み: 個人開発者が同じ土俵で汎用AI競争するのは難しい
- Gauth: AI Study Companion / ByteDance系と推定、提供元詳細は未確認 / AI学習支援 / 強み: 宿題・学習の明確なユースケース / 弱み: 教育コンテンツ品質と地域対応が差別化要因
- ReelShort / NewLeaf Publishing / 短尺ドラマ配信 / 強み: 縦型動画とアプリ内課金の相性 / 弱み: コンテンツ制作コストが高い
