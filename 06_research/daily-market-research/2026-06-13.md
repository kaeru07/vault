# 市場調査ログ 2026-06-13

## 今日の結論
次に見るべきは「AI日常アシスタントの不満を、用途特化アプリに切り出せる領域」。
ChatGPT は App Store / Google Play の生産性カテゴリで依然強く、汎用AI需要は確認できる。
一方で、画像生成の意図ズレ、回答の回りくどさ、無料枠・課金体験への不満が観察された。
個人開発では、汎用AIそのものより「短い作業を確実に終わらせる小型ツール」に寄せる方が収益化しやすい。

## 注目ジャンル
- AI日常アシスタント / ChatGPT が Google Play で 1B+ downloads、App Store でも Productivity 上位 / App Store, Google Play
- ショート動画編集・字幕生成 / CapCut が 1B+ downloads、テンプレ・自動字幕・背景削除が収益導線になりやすい / Google Play, TikTok, YouTube Shorts
- AIコーディングエージェント / Codex、Claude Code、Cursor、Copilot 系が「複数エージェント管理」へ移行 / AIニュース, GitHub, 開発者市場
- AIブラウザ・画面理解 / Gemini in Chrome や ChatGPT の画面共有機能が、閲覧支援・要約・比較に拡張 / Google, Chrome, Android
- 学習支援AI / App Storeレビューで統計・宿題・説明用途が強く観察された / App Store, Google Play
- クリエイター向けAI編集 / CapCut と Meta Edits の競争により、SNS投稿補助ツールの需要が継続 / App Store, Google Play, Instagram

## 競合アプリ
- ChatGPT / OpenAI / 音声、画像生成、写真入力、学習、業務相談をまとめた汎用AI / 強み: 圧倒的認知とストア上位 / 弱み: 回答が回りくどい、画像編集が意図通りでないという不満あり
- CapCut / ByteDance / 自動字幕、背景削除、テンプレ、4K書き出し対応の動画編集 / 強み: Shorts・TikTok制作との相性 / 弱み: Google Play評価は 3.5 と低めで品質不満の余地あり
- Edits / Meta Platforms / Instagram連携の短尺動画編集アプリ / 強み: Instagram運用データと直結 / 弱み: CapCut後発で差別化が課題
- Cursor / Anysphere / AI前提のコードエディタ / 強み: 開発者課金単価が高い / 弱み: 料金・利用量制限への不満が出やすい領域
- Gemini / Google / ChromeやAndroidに統合されるAIアシスタント / 強み: ブラウザ・Gmail・Calendar・Maps連携 / 弱み: 個人開発者が正面衝突すると勝ちにくい

## ユーザー不満
- ChatGPT の画像生成・画像編集が指示とズレる、細部修正に時間がかかるという不満あり。情報源: Google Playレビュー、内容の真偽は未確認。
- ChatGPT の無料枠終了後に体験が急に悪化したと感じるレビューあり。情報源: Google Playレビュー、内容の真偽は未確認。
- ChatGPT がすぐ答えず、回りくどいやり取りになるという不満あり。情報源: App Storeレビュー、内容の真偽は未確認。
- CapCut 系動画編集は多機能化により、初心者には「どこを押せばよいか分からない」余地がある。これはストア機能量からの推測で未確認。
- AIコーディングエージェントは実務モバイル開発で成功率がまだ低いという研究報告があり、過信への不満が起きやすい。

## 収益化ヒント
- 月額課金: 「画像修正だけ」「字幕だけ」「学習問題化だけ」など、汎用AIの一部を確実に実行する小型SaaS。
- 広告収益: Shorts台本、字幕、サムネ案、投稿文を無料生成し、回数制限後に広告表示。
- note / 動画教材: 「ChatGPTで失敗しがちな画像修正をプロンプト化する」「CapCut時短テンプレ集」などの販売。
- アプリ内課金: 無料で数件生成、有料で一括処理・履歴保存・ブランドプリセット解放。
- アフィリエイト: AIツール比較、動画編集テンプレ、開発者向けAIエージェント運用ノウハウへの誘導。
