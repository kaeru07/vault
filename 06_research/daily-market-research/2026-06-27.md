# 市場調査ログ 2026-06-27

## 今日の結論
次に見るべきは「AIエージェントを一般ユーザーが安全に委任できる小型ワークフロー」と「ショート動画向けAI編集」の課金導線。  
App Store / Google Play では ChatGPT の大規模定着が続き、単体AIチャットの正面勝負は難しい。  
一方で、Perplexity Comet や MCP 周辺の動きから、ブラウザ・メール・調査・購入補助などの委任型UIに余地がある。  
個人開発では、巨大AIの代替ではなく「特定ジャンルの作業を短く終わらせる補助ツール」に絞る方が収益化に近い。

## 注目ジャンル
- AIブラウザ / 調査代行 / Perplexity Comet、ChatGPT、Chrome拡張
- AIエージェント権限管理 / 企業・個人の安全な自動化需要 / MCP、A2A、Arcade.dev
- ショート動画AI編集 / SNS投稿の継続需要とテンプレ課金 / CapCut、TikTok、YouTube Shorts
- 学習・説明AI / ChatGPTレビューで学習用途が目立つ / App Store、Google Play
- マイクロSaaS型の業務自動化 / 請求、議事録、メール返信など狭い反復作業 / Webアプリ、Chrome拡張
- AI生成ショートドラマ / 低コスト大量制作と視聴維持の実験余地 / TikTok、YouTube Shorts

## 競合アプリ
- ChatGPT / OpenAI / App Storeで「Productivity #1」、Google Playで1B+ downloads表示 / 強み: 汎用性・認知・音声・画像生成 / 弱み: 特定業務の完了体験はまだ汎用的
- Perplexity Comet / Perplexity AI / ブラウザ内アシスタントでメール、買い物、調査、Web操作を委任 / 強み: 検索と行動の統合 / 弱み: セキュリティ不安や利用習慣の壁
- CapCut / ByteDance / AI動画生成、字幕、背景除去、テンプレ、音声系ツールを統合 / 強み: ショート動画制作導線が強い / 弱み: 汎用編集アプリなので業界特化テンプレは差別化余地あり
- Arcade.dev / Arcade / AIエージェントの認可・監査・実行制御 / 強み: エージェント普及時の基盤需要 / 弱み: 個人開発が直接競合するには重い
- Focus Friend / 個人開発系プロダクトとして未確認 / 一時的なApp Store上位化が二次情報で報じられた / 強み: SNS・著名人導線で急伸可能 / 弱み: 継続課金の強さは未確認

## ユーザー不満
- ChatGPTのApp Storeレビューでは「以前より直接答えず、余計な対話が増えた」「遅い・クラッシュする」といった不満が観察できた。
- AIブラウザ系は便利だが、メール・買い物・金融情報まで委任することへのセキュリティ不安が残る。
- 動画編集AIは機能が多すぎ、初心者や業種別ユーザーには「何から作るか」が重い可能性がある。
- AI生成ショート動画は伸びる一方で、視聴者から「AI slop」扱いされるリスクがある。情報源は二次情報中心で一部未確認。
- Google Play課金手数料の変更は開発者に有利な可能性があるが、地域・時期・条件が複雑で個人開発者への実利は未確認。

## 収益化ヒント
- 業種特化テンプレ課金: 不動産、飲食、美容室、士業向けに「投稿文 + 縦動画 + サムネ」を一括生成。
- Chrome拡張の月額課金: Gmail返信、求人応募、リサーチ要約など、毎日使う小さい委任タスクに絞る。
- note / 動画販売: 「AIで毎日ショート動画を作る運用テンプレ」「CapCut自動化レシピ」を教材化。
- 広告収益: 無料のAI字幕・タイトル生成・台本診断ツールでSEO流入を取り、上位版を課金へ誘導。
- アフィリエイト: AIツール比較、動画生成ツール比較、MCPホスティング比較から登録導線を作る。

## 作れそうなアプリ案
- ショート動画台本リライター: TikTok / Shorts向けに冒頭3秒、字幕、タイトル、説明文を生成。実装難度: 低
- 業種別CapCutテンプレ設計ツール: 店舗ジャンルを入れると構成、字幕、素材リスト、投稿文を出す。実装難度: 低
- AIブラウザ用タスク監査メモ: ユーザーがAIに委任した操作を記録し、リスク分類するローカルWebアプリ。実装難度: 中
- 学習AIプロンプトノート: 苦手科目を入れると「説明、例題、確認テスト」を毎日生成して保存。実装難度: 中
- MCPツール棚卸しダッシュボード: 自分のMCPサーバーの権限、外部接続、危険操作を可視化。実装難度: 高
