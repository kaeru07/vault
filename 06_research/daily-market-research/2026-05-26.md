# 市場調査ログ 2026-05-26

## 今日の結論
次に見るべきは、Google Play の AI 検索導線「Ask Play / Gemini からのアプリ推薦」が個人開発アプリの流入に与える影響。  
App Store の米国 Productivity 上位は ChatGPT / Claude / Gemini / Meta AI / Grok が並び、汎用 AI 助手は寡占化している。  
個人開発では「汎用チャット」ではなく、検索・動画化・業務代行・縦型業務の小さな痛みに寄せた方が収益化に近い。  
短期試作は、AI エージェント実行ログ、Shorts 再利用、App Store/Google Play ASO 監視の 3 系統が有望。

## 競合アプリ
- ChatGPT / OpenAI / App Store Productivity 上位の汎用 AI アシスタント / 強み: 認知・汎用性・課金導線 / 弱み: 個別業務への深い最適化は別アプリ余地あり
- Claude / Anthropic / 文章・思考・コーディング支援 / 強み: 長文処理と開発者支持 / 弱み: モバイル単体の業務実行導線は限定的
- Google Gemini / Google / Gemini、Android、Play、Workspace 連携 / 強み: OS・検索・ストア導線を持つ / 弱み: 個人開発者は Google 依存と差別化難度が高い
- Google Antigravity / Google / エージェント中心の開発環境 / 強み: Android・Firebase・Gemini API 連携 / 弱み: 競争が激しく、単純な開発支援ツールは埋もれやすい
- Opus Clip 系の動画再利用ツール / 複数提供元 / 長尺動画から短尺クリップ生成 / 強み: 収益化導線が明確 / 弱み: 類似ツール過多で価格競争になりやすい

## 収益化ヒント
- Google Play / App Store の検索順位、説明文、スクリーンショットを毎日監視する ASO レポートを月額課金にする
- 長尺動画・記事・note を Shorts / TikTok / Reels 用に変換するツールを、従量課金またはクレジット制で提供する
- Claude Code / Codex / Antigravity 向けに、実行ログ・費用・失敗理由を可視化する開発者向け SaaS を作る
- AI ツール比較記事、動画、note からアフィリエイト・有料テンプレ・プロンプト集へ誘導する
- 縦型業務向けに「AI が下書き、ユーザーが承認」の小規模 B2B サブスクを作る

## 作れそうなアプリ案
- AI App Radar: App Store / Google Play の AI・Productivity 順位と説明文変化を監視するダッシュボード。実装難度: 中
- Shorts Repurposer Lite: URL または原稿から 60 秒台本、字幕、投稿文を生成する小型ツール。実装難度: 低
- Agent Cost Logger: Codex / Claude Code / Gemini API の実行ログ、失敗、概算コストを記録する開発者向けツール。実装難度: 中
- Product Hunt Gap Finder: Product Hunt / Reddit の不満投稿から、類似ツール過多と未充足ニーズを分類する調査ツール。実装難度: 中
- Vertical Admin AI: セラピスト、講師、士業など特定職種向けの予約・メモ・請求下書き支援。実装難度: 高
