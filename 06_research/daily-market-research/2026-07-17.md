# 市場調査ログ 2026-07-17

## 今日の結論
次に見るべきは、汎用AIチャット単体ではなく「AIエージェント運用」「ショート動画広告生成」「サブスク継続率改善」の周辺ツールである。  
App Store / Google Play では ChatGPT、Gemini、Claude が引き続き上位圏に入り、AIアプリ需要は一過性ではない。  
TikTok Shop 周辺では AI生成動画と広告運用代行が伸びており、個人開発でも小規模EC向けツールに収益化余地がある。  
Google Play と App Store の課金・分析機能改善により、サブスク型アプリの実験コストは下がりつつある。

## 記事本文
今日の観測で最も重要なのは、AIアプリ市場が「便利なチャット」から「成果物を出す作業単位」へ移っている点だ。OpenAI は ChatGPT Work を発表し、長時間の調査、資料、Webアプリ作成までを担うエージェント型の作業面を強めている。Anthropic 側も Claude Code や Claude Tag の文脈で、個人の補助ツールからチームに常駐する作業者へ寄せている。個人開発者にとっては、基盤モデルそのものと競うより、特定業務の入口、監視、承認、成果物整形を担う小さなアプリの方が現実的だ。

モバイル市場では、Appfigures のランキングで ChatGPT、Google Gemini、Claude が上位に見えており、AIアシスタントが通常のSNS、動画、地図、ECアプリと同じ棚に並んでいる。特に ChatGPT はモバイル課金規模の大きさが繰り返し報じられており、Claude も収益成長が注目されている。ただし、これは大手ブランドの勝利であって、個人開発者が汎用AIチャットを作れば勝てるという意味ではない。むしろ、AI出力をアプリ内の具体的な行動に変換する「履歴整理」「比較表化」「投稿文生成」「レビュー返信」「動画台本化」のような狭い導線が狙い目になる。

SNSと広告側では、TikTok が Symphony Agent や Creative Center を強化し、TikTok Shop では AI生成動画や管理型広告運用の話題が増えている。AI動画が増えすぎることで信頼性やスパム懸念も出ているが、裏返すと、ブランドや小規模販売者は「AIで量産したいが、虚偽表現や規約違反は避けたい」という不満を抱え始めている。ここは個人開発でも、商品ページから複数の縦型動画台本を作る、禁止表現をチェックする、投稿後の反応を整理する、といった小さなSaaSやテンプレ販売につなげやすい。

プラットフォーム面では、Google Play が AIによるローカライズ、キーワード別ストア掲載、購読継続の分析を強め、Apple もグループ購読、バンドル、Retention Messaging などを用意している。これは、アプリを作って終わりではなく、価格、継続率、解約時オファーを毎週改善する運用型の収益化が重要になるという合図だ。Claude Code / Codex で短期試作するなら、まずは1つのジャンルに絞り、AI生成、公開前チェック、収益指標の記録までを一気通貫にする軽量ツールを作るのが次の一歩になる。

## 競合アプリ
- ChatGPT / OpenAI / モバイルAIアシスタント、音声、画像、作業エージェント化 / ブランド力と課金導線が強い / 汎用すぎて業務特化の細部は弱点になり得る
- Claude / Anthropic / 長文、コーディング、チーム文脈のAI支援 / 開発者・業務利用で強い / 一般消費者向けの導線はChatGPTほど広くない可能性
- Google Gemini / Google / Android、検索、Workspaceとの統合 / 配布面とOS統合が強い / エージェント型コーディングでは競合に遅れとの二次情報あり
- CapCut / ByteDance / ショート動画編集とテンプレ生成 / TikTok連携とクリエイター浸透が強い / AI生成動画の品質・透明性への不満が出やすい
- ChatOn / AIBY系と推測 / 複数AIモデルを束ねるアシスタント / 乗り換えコストを下げる訴求が強い / モデル本体の差別化は難しい
