## AI業界の変化
- 短期: Coding agent は「補完」ではなく、並列サブエージェント、テスト実行、PR準備まで含む開発運用レイヤーになっている。
- 短期: Gemini Spark、Claude dynamic workflows、Codex enterprise controls から、常駐型・非同期型エージェントが主戦場化。
- 中期: MCP/API 経由の機械アクセスが増え、SaaS はユーザー席課金から消費量課金へ移行しやすい。
- 中期: 金融・法務・サイバー防御など高単価領域で frontier model の導入が先行し、個人開発者は周辺ツールや監査・可視化で参入余地がある。

## 個人開発への影響
1. 今週は「単発チャットUI」より、定期実行・バックグラウンド実行・承認待ちを持つエージェントUIを優先して設計する。
2. MCP対応は収益化に近い。既存SaaSや個人ツールを「AIから呼べるAPI」にするだけでも価値が出やすい。
3. Claude Code / Codex 前提で、テスト・lint・スクショ検証まで自動実行できる開発テンプレを整備する。
4. 料金設計は月額固定だけでなく、実行回数・API呼び出し・生成時間の上限を必ず入れる。
5. 動画生成・要約・日次ブリーフ系は Gemini Omni / Spark の影響で競争が強まるため、麻雀・将棋・学習など特化領域に寄せる。

## 収益化への影響
- SaaS: MCP/API消費量課金の流れは、小規模SaaSにも「AIエージェント用プラン」を作る根拠になる。
- 課金: Claude / Codex / Gemini とも高性能モデルはトークン・実行時間コストが重い。個人開発では無料枠の濫用対策が必須。
- 配信: Gemini Omni の動画生成強化により、Shorts/TikTok向けの量産補助ツールは参入しやすいが差別化が必要。
- 広告: ChatGPT 個人金融や Gemini Daily Brief のような高頻度接触面が増え、個人開発者はニッチな日次レポート・通知アプリで広告/課金を組み合わせやすい。
