# AIニュース 2026-06-02

## 今日重要だったニュース
- Microsoft Build 2026 開幕 / 公式  
  Microsoft Build が 6月2日 PT に開始。Microsoft Agent Framework、Foundry、AI 開発者向けツールが中心テーマ。個人開発への影響度: 高
- OpenAI frontier models と Codex が AWS で一般提供 / 公式  
  OpenAI models と Codex が Amazon Bedrock 経由で利用可能に。Codex は「週 500 万人以上が利用」とされ、企業導入の摩擦が下がる。個人開発への影響度: 高
- Anthropic が秘密裏に S-1 を提出 / 公式  
  Anthropic が 6月1日に IPO に向けた Form S-1 草案を SEC に秘密提出。Claude の商用化・資金調達競争が一段進んだ。個人開発への影響度: 中
- Claude Opus 4.8 と Claude Code dynamic workflows / 公式  
  Opus 4.8 は同価格で提供、Claude Code は大規模タスク向け dynamic workflows を追加。長時間・複数サブエージェント型開発が進む。個人開発への影響度: 高
- Google Gemini 3.5 Flash / Antigravity 2.0 / Managed Agents / 公式  
  Google I/O 2026 の発表が継続的に展開。Gemini 3.5 Flash、Antigravity 2.0、Gemini API Managed Agents は小規模開発者にも実装基盤になり得る。個人開発への影響度: 高

## AI業界の変化
- 短期: AI コーディングは「IDE 補助」から「クラウド・ローカル・企業環境に入る開発エージェント」へ移行。OpenAI on AWS、Claude Code dynamic workflows、Google Antigravity が同じ方向を向いている。
- 短期: 従量・クレジット課金への圧力が強まっている。GitHub Copilot は AI credits を中心にプラン説明を更新しており、無制限型サブスクは持続しにくい。
- 中期: 主要モデル企業はモデル単体より「配布面」を競っている。AWS、Google Cloud、Microsoft、IDE、Search、Workspace、Android への統合が差別化要因。
- 中期: AI 生成コードの速度向上に対し、テスト・CI・ロールバック・監視が収益化の制約になっている。AI で作るより、壊さず出す仕組みが重要。

- OpenAI: ChatGPT では GPT-4.5 が 2026-06-27、OpenAI o3 が 2026-08-26 に退役予定。API 変更はなし。
- Gemini: Gemini 3.5 Flash は Antigravity、Gemini API、Google AI Studio、Android Studio、Search AI Mode に展開。
- Gemini: Managed Agents は Gemini API から単一 API 呼び出しで隔離 Linux 環境つきエージェントを起動できる設計。

## 個人開発への影響
- 今週は AI コーディング利用料を棚卸しする。Codex、Claude Code、Copilot、Gemini/Antigravity の月額・クレジット・API 単価を比較し、1 プロジェクトあたりの上限を決める。
- 新規アプリは「AI が作る」前提で、最初からテスト、lint、型チェック、ロールバック手順を入れる。生成速度だけ上げると修正コストが後段に移る。
- Codex / Claude Code / Antigravity の 3 系統を小さな同一タスクで比較する。候補は LP 生成、既存コード修正、テスト追加、README 作成。
- 個人開発では AWS / Google Cloud / GitHub 連携が強いツールを優先確認する。企業向け導入と同じ配布経路が、後の B2B 販売に効く。
- Google AI Studio の Android / Play Console 連携は、学習アプリ・麻雀・将棋・ショート動画補助アプリの試作に向く。

## 収益化への影響
- SaaS: OpenAI on AWS と Codex on Bedrock により、企業向け「既存 AWS 環境で動く AI 業務ツール」の販売障壁が下がる。
- 課金: Copilot / Claude / Google AI Ultra などでクレジット・上限設計が前面化。個人開発者も自サービスに利用上限、追加課金、ジョブ単価表示を入れるべき。
- 配信: Google Search の generative UI / mini app 化は、SEO だけに依存する集客のリスク。検索結果内で完結する体験に奪われる可能性がある。
- 広告: ChatGPT の広告テストは 2026年2月時点の公式情報。今日の新規発表ではないが、無料 AI サービスの広告モデル化は継続監視。
