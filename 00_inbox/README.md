# 00_inbox — 未分類メモ置き場

分類が決まる前の思いつき・断片メモを置く場所。**完成度を気にせず、まずここに書く**。

## 入れていいもの

- 一行メモ・思いつき・気付き
- どのフォルダに置くべきか判断がつかないメモ
- 後でアイデアに育てたい雑な記録
- ニュース・記事のリンクと一言コメント

## 入れないもの

- 機密情報（API キー / .env / 個人情報など）→ vault 全体で禁止
- 明確に分類先が決まっているノート → 最初から該当フォルダへ
- 完成された仕様・レビュー → `02_apps/` / `04_reviews/` へ

## 運用ルール

- まず `inbox.md` に一行追記でよい。後から独立 md に切り出す
- 育ったメモは適切なフォルダへ移動する:
  - アプリ仕様 → `02_apps/`
  - プロンプト → `03_prompts/`
  - 収益化アイデア → `05_monetization/`
  - 市場調査 → `06_research/`
- 移動時は Obsidian の機能で行う（`[[link]]` が自動追従）
- inbox は週次で見直し、不要なものは消す or 該当フォルダへ動かす

## ファイル

- `inbox.md` — メイン雑記。一行メモはここ
- 大きなテーマは独立 md を作る（`<topic>.md`）
