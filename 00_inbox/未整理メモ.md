# 未整理メモ

> 一時メモ置き場。ChatGPT / Claude / Codex からの断片メモ・思いつき・気付きをまず**ここに投げ込む**。後で整理する前提なので完成度は気にしない。

## 使い方

- **書くのは一行でいい**。完成度は無視
- 出典は分かる範囲で：`(ChatGPT)` `(Claude)` `(Codex)` `(自分)` `(URL)` などを語尾に
- 育ったメモは独立 md に切り出す
- 分類が決まったら該当フォルダへ移動する：
  - アプリ仕様 → `02_apps/`
  - プロンプト → `03_prompts/`
  - 収益化アイデア → `05_monetization/`
  - 市場調査 → `06_research/`
- 移動後はリンク `[[note-name]]` をここに残してもよい

## 書いていいもの

- 思いつき・仮説・違和感
- 他人のツイートや記事のリンクと一言コメント
- ChatGPT / Claude の応答の中で「これ後で使える」と思った断片
- 朝起きた瞬間に浮かんだアイデア

## 書かないもの

- API キー・`.env`・認証情報・個人情報（vault 全体で禁止）
- 完成された仕様 → 最初から `02_apps/` 等へ
